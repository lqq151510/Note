# RabbitMQ 消息转换器（MessageConverter）

消息转换器是 Spring AMQP 的核心组件，负责在 **Java 对象**​ 和 **RabbitMQ 消息**​ 之间进行转换。Spring Boot 提供了多种内置转换器，也支持自定义转换器。

## 一、 内置消息转换器

### 1. 常用转换器概览

|转换器类|说明|适用场景|
|---|---|---|
|`SimpleMessageConverter`|默认转换器，支持基本类型|简单字符串、字节数组|
|`Jackson2JsonMessageConverter`|JSON 转换器|JSON 对象序列化|
|`MarshallingMessageConverter`|XML 转换器|XML 序列化|
|`SerializerMessageConverter`|序列化转换器|Java 原生序列化|
|`ContentTypeDelegatingMessageConverter`|代理转换器|根据 ContentType 动态选择|

### 2. 默认转换器配置

```
@Configuration
public class DefaultConverterConfig {
    
    /**
     * 默认转换器（不配置时的行为）
     * SimpleMessageConverter 支持：
     * - String → byte[]
     * - byte[] → String
     * - Serializable → byte[]（Java序列化）
     */
    @Bean
    public MessageConverter simpleMessageConverter() {
        return new SimpleMessageConverter();
    }
    
    /**
     * 使用示例
     */
    @Component
    public class SimpleConverterExample {
        
        @RabbitListener(queues = "simple.queue")
        public void handleString(String message) {
            // 接收字符串
            System.out.println("字符串消息: " + message);
        }
        
        @RabbitListener(queues = "simple.queue2")
        public void handleBytes(byte[] message) {
            // 接收字节数组
            System.out.println("字节消息: " + new String(message));
        }
        
        @RabbitListener(queues = "simple.queue3")
        public void handleSerializable(SerializableObject obj) {
            // 接收可序列化对象（需要实现Serializable）
            System.out.println("序列化对象: " + obj);
        }
    }
}

## 二、 Jackson2JsonMessageConverter（推荐）

### 1. 基础配置

```
@Configuration
public class JacksonConverterConfig {
    
    /**
     * JSON 消息转换器（推荐）
     */
    @Bean
    public MessageConverter jsonMessageConverter() {
        // 创建 Jackson2JsonMessageConverter
        Jackson2JsonMessageConverter converter = new Jackson2JsonMessageConverter();
        
        // 设置 Jackson ObjectMapper
        ObjectMapper objectMapper = new ObjectMapper();
        
        // 配置 ObjectMapper
        objectMapper.configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);
        objectMapper.configure(SerializationFeature.WRITE_DATES_AS_TIMESTAMPS, false);
        objectMapper.registerModule(new JavaTimeModule());  // 支持 Java 8 时间
        objectMapper.setSerializationInclusion(JsonInclude.Include.NON_NULL);  // 忽略 null
        
        converter.setObjectMapper(objectMapper);
        
        return converter;
    }
    
    /**
     * 在 RabbitTemplate 中使用
     */
    @Bean
    public RabbitTemplate rabbitTemplate(ConnectionFactory connectionFactory) {
        RabbitTemplate template = new RabbitTemplate(connectionFactory);
        template.setMessageConverter(jsonMessageConverter());
        return template;
    }
    
    /**
     * 在监听器容器工厂中使用
     */
    @Bean
    public SimpleRabbitListenerContainerFactory rabbitListenerContainerFactory(
            ConnectionFactory connectionFactory) {
        
        SimpleRabbitListenerContainerFactory factory = new SimpleRabbitListenerContainerFactory();
        factory.setConnectionFactory(connectionFactory);
        factory.setMessageConverter(jsonMessageConverter());
        
        return factory;
    }
}

## 三、 自定义消息转换器

### 1. 实现 MessageConverter 接口

```
@Component
public class CustomMessageConverter implements MessageConverter {
    
    private static final String CONTENT_TYPE_JSON = "application/json";
    private static final String CONTENT_TYPE_XML = "application/xml";
    private static final String CONTENT_TYPE_PROTOBUF = "application/x-protobuf";
    
    private final ObjectMapper objectMapper = new ObjectMapper();
    private final JAXBContext jaxbContext;
    
    public CustomMessageConverter() throws JAXBException {
        this.jaxbContext = JAXBContext.newInstance(User.class, Order.class);
    }
    
    @Override
    public Message toMessage(Object object, MessageProperties messageProperties) 
            throws MessageConversionException {
        
        try {
            byte[] body;
            String contentType = messageProperties.getContentType();
            
            if (object instanceof String) {
                // 字符串处理
                body = ((String) object).getBytes(StandardCharsets.UTF_8);
                messageProperties.setContentType(MessageProperties.CONTENT_TYPE_TEXT_PLAIN);
                messageProperties.setContentEncoding("UTF-8");
                
            } else if (CONTENT_TYPE_JSON.equals(contentType) || contentType == null) {
                // JSON 序列化（默认）
                body = objectMapper.writeValueAsBytes(object);
                messageProperties.setContentType(CONTENT_TYPE_JSON);
                messageProperties.setContentEncoding("UTF-8");
                
                // 添加类型信息
                messageProperties.getHeaders().put("__TypeId__", object.getClass().getName());
                
            } else if (CONTENT_TYPE_XML.equals(contentType)) {
                // XML 序列化
                StringWriter writer = new StringWriter();
                Marshaller marshaller = jaxbContext.createMarshaller();
                marshaller.marshal(object, writer);
                body = writer.toString().getBytes(StandardCharsets.UTF_8);
                messageProperties.setContentType(CONTENT_TYPE_XML);
                
            } else if (CONTENT_TYPE_PROTOBUF.equals(contentType)) {
                // Protobuf 序列化
                if (object instanceof User) {
                    UserProto.User userProto = convertToProto((User) object);
                    body = userProto.toByteArray();
                } else {
                    throw new MessageConversionException("不支持的Protobuf类型: " + object.getClass());
                }
                messageProperties.setContentType(CONTENT_TYPE_PROTOBUF);
                
            } else {
                throw new MessageConversionException("不支持的Content-Type: " + contentType);
            }
            
            return new Message(body, messageProperties);
            
        } catch (Exception e) {
            throw new MessageConversionException("消息转换失败", e);
        }
    }
    
    @Override
    public Object fromMessage(Message message) throws MessageConversionException {
        try {
            MessageProperties properties = message.getMessageProperties();
            String contentType = properties.getContentType();
            byte[] body = message.getBody();
            
            if (body == null || body.length == 0) {
                return null;
            }
            
            // 根据Content-Type选择反序列化方式
            if (MessageProperties.CONTENT_TYPE_TEXT_PLAIN.equals(contentType)) {
                // 文本处理
                return new String(body, StandardCharsets.UTF_8);
                
            } else if (CONTENT_TYPE_JSON.equals(contentType)) {
                // JSON 反序列化
                String typeId = (String) properties.getHeaders().get("__TypeId__");
                if (typeId != null) {
                    Class<?> targetClass = Class.forName(typeId);
                    return objectMapper.readValue(body, targetClass);
                } else {
                    // 尝试自动推断类型
                    return objectMapper.readValue(body, Object.class);
                }
                
            } else if (CONTENT_TYPE_XML.equals(contentType)) {
                // XML 反序列化
                String xml = new String(body, StandardCharsets.UTF_8);
                Unmarshaller unmarshaller = jaxbContext.createUnmarshaller();
                return unmarshaller.unmarshal(new StringReader(xml));
                
            } else if (CONTENT_TYPE_PROTOBUF.equals(contentType)) {
                // Protobuf 反序列化
                String typeId = (String) properties.getHeaders().get("__TypeId__");
                if (User.class.getName().equals(typeId)) {
                    UserProto.User userProto = UserProto.User.parseFrom(body);
                    return convertFromProto(userProto);
                }
                throw new MessageConversionException("未知的Protobuf类型: " + typeId);
                
            } else {
                // 默认按JSON处理
                return objectMapper.readValue(body, Object.class);
            }
            
        } catch (Exception e) {
            throw new MessageConversionException("消息解析失败", e);
        }
    }
    
    // Protobuf 转换方法
    private UserProto.User convertToProto(User user) {
        return UserProto.User.newBuilder()
                .setId(user.getId())
                .setName(user.getName())
                .setEmail(user.getEmail())
                .build();
    }
    
    private User convertFromProto(UserProto.User userProto) {
        User user = new User();
        user.setId(userProto.getId());
        user.setName(userProto.getName());
        user.setEmail(userProto.getEmail());
        return user;
    }
}
```

### 2. 自定义 JSON 转换器（带加密）

```
@Component
public class EncryptedJsonMessageConverter extends AbstractJsonMessageConverter {
    
    private final ObjectMapper objectMapper = new ObjectMapper();
    private final CryptoService cryptoService;
    
    public EncryptedJsonMessageConverter(CryptoService cryptoService) {
        this.cryptoService = cryptoService;
        objectMapper.configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);
    }
    
    @Override
    protected Message createMessage(Object object, MessageProperties messageProperties) 
            throws MessageConversionException {
        
        try {
            // 序列化为 JSON
            String json = objectMapper.writeValueAsString(object);
            
            // 加密
            String encrypted = cryptoService.encrypt(json);
            
            // 设置消息属性
            messageProperties.setContentType(MessageProperties.CONTENT_TYPE_JSON);
            messageProperties.setContentEncoding("UTF-8");
            messageProperties.setHeader("encrypted", true);
            messageProperties.setHeader("algorithm", cryptoService.getAlgorithm());
            
            byte[] body = encrypted.getBytes(StandardCharsets.UTF_8);
            return new Message(body, messageProperties);
            
        } catch (Exception e) {
            throw new MessageConversionException("加密消息转换失败", e);
        }
    }
    
    @Override
    public Object fromMessage(Message message) throws MessageConversionException {
        try {
            MessageProperties properties = message.getMessageProperties();
            
            // 检查是否加密
            Boolean encrypted = (Boolean) properties.getHeaders().get("encrypted");
            String bodyStr = new String(message.getBody(), StandardCharsets.UTF_8);
            
            if (encrypted != null && encrypted) {
                // 解密
                bodyStr = cryptoService.decrypt(bodyStr);
            }
            
            // 获取类型信息
            String typeId = (String) properties.getHeaders().get("__TypeId__");
            if (typeId != null) {
                Class<?> targetClass = Class.forName(typeId);
                return objectMapper.readValue(bodyStr, targetClass);
            }
            
            // 默认反序列化
            return objectMapper.readValue(bodyStr, Object.class);
            
        } catch (Exception e) {
            throw new MessageConversionException("解密消息转换失败", e);
        }
    }
    
    /**
     * 发送时添加类型信息
     */
    public void setTypeId(MessageProperties properties, Class<?> clazz) {
        properties.getHeaders().put("__TypeId__", clazz.getName());
    }
}

## 四、 ContentTypeDelegatingMessageConverter

### 1. 多格式支持转换器

```
@Configuration
public class DelegatingConverterConfig {
    
    @Bean
    public ContentTypeDelegatingMessageConverter delegatingMessageConverter() {
        ContentTypeDelegatingMessageConverter converter = 
            new ContentTypeDelegatingMessageConverter();
        
        // 注册不同Content-Type的转换器
        converter.addDelegate("application/json", jsonMessageConverter());
        converter.addDelegate("application/xml", xmlMessageConverter());
        converter.addDelegate("text/plain", textMessageConverter());
        converter.addDelegate("application/x-java-serialized-object", javaMessageConverter());
        converter.addDelegate("application/x-protobuf", protobufMessageConverter());
        
        // 设置默认转换器
        converter.setDefaultConverter(jsonMessageConverter());
        
        return converter;
    }
    
    @Bean
    public MessageConverter jsonMessageConverter() {
        Jackson2JsonMessageConverter converter = new Jackson2JsonMessageConverter();
        converter.getObjectMapper().configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);
        return converter;
    }
    
    @Bean
    public MessageConverter xmlMessageConverter() {
        MarshallingMessageConverter converter = new MarshallingMessageConverter();
        
        Jaxb2Marshaller marshaller = new Jaxb2Marshaller();
        marshaller.setPackagesToScan("com.example.model");
        
        converter.setMarshaller(marshaller);
        converter.setUnmarshaller(marshaller);
        
        return converter;
    }
    
    @Bean
    public MessageConverter textMessageConverter() {
        return new SimpleMessageConverter();
    }
    
    @Bean
    public MessageConverter javaMessageConverter() {
        return new SerializerMessageConverter();
    }
    
    @Bean
    public MessageConverter protobufMessageConverter() {
        return new ProtobufMessageConverter();
    }
    
    /**
     * 使用示例
     */
    @Component
    public class DelegatingConverterExample {
        
        @Autowired
        private RabbitTemplate rabbitTemplate;
        
        public void sendDifferentFormats() {
            User user = new User(1, "张三", "zhangsan@example.com");
            
            // 发送 JSON
            MessageProperties jsonProps = new MessageProperties();
            jsonProps.setContentType("application/json");
            rabbitTemplate.convertAndSend("format.exchange", "json", user, 
                new MessagePostProcessor() {
                    @Override
                    public Message postProcessMessage(Message message) {
                        message.getMessageProperties().setContentType("application/json");
                        return message;
                    }
                });
            
            // 发送 XML
            rabbitTemplate.convertAndSend("format.exchange", "xml", user, 
                message -> {
                    message.getMessageProperties().setContentType("application/xml");
                    return message;
                });
            
            // 发送文本
            rabbitTemplate.convertAndSend("format.exchange", "text", 
                "纯文本消息", 
                message -> {
                    message.getMessageProperties().setContentType("text/plain");
                    return message;
                });
        }
        
        /**
         * 接收多种格式的消息
         */
        @RabbitListener(queues = "format.queue")
        public void handleMultipleFormats(@Payload Object payload,
                                         @Header(AmqpHeaders.CONTENT_TYPE) String contentType) {
            
            System.out.println("Content-Type: " + contentType);
            System.out.println("消息: " + payload);
            System.out.println("类型: " + payload.getClass());
        }
    }
}

## 五、 消息转换器在生产者中的应用

### 1. RabbitTemplate 配置

@Configuration
public class ProducerConverterConfig {
    
    @Bean
    public RabbitTemplate jsonRabbitTemplate(ConnectionFactory connectionFactory) {
        RabbitTemplate template = new RabbitTemplate(connectionFactory);
        template.setMessageConverter(jsonMessageConverter());
        
        // 消息确认回调
        template.setConfirmCallback((correlationData, ack, cause) -> {

            } else {
                System.out.println("消息发送失败: " + cause);

        return template;
    }
    
    @Bean
    public MessageConverter jsonMessageConverter() {

    public class ProducerExample {

                jsonRabbitTemplate.convertAndSend("batch.exchange", "batch.key", user);
            });
        }

         * 发送延迟消息

                message1 -> {
                    message1.getMessageProperties().setHeader("x-delay", delayMillis);
                    return message1;
                });
        }
    }

### 2. 消息后处理器

```
@Component
public class MessagePostProcessorExample {
    
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    /**
     * 添加通用 headers
     */
    public MessagePostProcessor addCommonHeaders() {
        return message -> {
            MessageProperties props = message.getMessageProperties();
            
            // 添加追踪信息
            props.setHeader("trace-id", UUID.randomUUID().toString());
            props.setHeader("app-id", "order-service");
            props.setHeader("send-time", System.currentTimeMillis());
            
            // 设置消息属性
            props.setDeliveryMode(MessageDeliveryMode.PERSISTENT);
            props.setPriority(1);
            props.setContentType(MessageProperties.CONTENT_TYPE_JSON);
            props.setContentEncoding("UTF-8");
            
            return message;
        };
    }
    
    /**
     * 添加过期时间
     */
    public MessagePostProcessor addExpiration(long ttlMillis) {
        return message -> {
            message.getMessageProperties().setExpiration(String.valueOf(ttlMillis));
            return message;
        };
    }
    
    /**
     * 添加消息类型
     */
    public MessagePostProcessor addTypeHeader(Class<?> type) {
        return message -> {
            message.getMessageProperties().setHeader("__TypeId__", type.getName());
            return message;
        };
    }
    
    /**
     * 使用示例
     */
    public void sendMessageWithPostProcessors() {
        User user = new User(1, "张三", "zhangsan@example.com");
        
        // 组合多个后处理器
        MessagePostProcessor combined = message -> {
            MessagePostProcessor p1 = addCommonHeaders();
            MessagePostProcessor p2 = addTypeHeader(User.class);
            MessagePostProcessor p3 = addExpiration(60000);
            
            message = p1.postProcessMessage(message);
            message = p2.postProcessMessage(message);
            message = p3.postProcessMessage(message);
            
            return message;
        };
        
        rabbitTemplate.convertAndSend("exchange", "key", user, combined);
    }

### 1. 压缩消息转换器

```
@Component
public class CompressingMessageConverter implements MessageConverter {
    
    private final MessageConverter delegate;
    private final int compressionThreshold = 1024; // 1KB
    
    public CompressingMessageConverter(MessageConverter delegate) {
        this.delegate = delegate;
    }
    
    @Override
    public Message toMessage(Object object, MessageProperties messageProperties) 
            throws MessageConversionException {
        
        Message message = delegate.toMessage(object, messageProperties);
        byte[] body = message.getBody();
        
        // 超过阈值时压缩
        if (body.length > compressionThreshold) {
            try {
                byte[] compressed = compress(body);
                message = new Message(compressed, message.getMessageProperties());
                message.getMessageProperties().setHeader("compressed", true);
            } catch (IOException e) {
                throw new MessageConversionException("压缩失败", e);
            }
        }
        
        return message;
    }
    
    @Override
    public Object fromMessage(Message message) throws MessageConversionException {
        MessageProperties properties = message.getMessageProperties();
        byte[] body = message.getBody();
        
        // 解压缩
        if (Boolean.TRUE.equals(properties.getHeaders().get("compressed"))) {
            try {
                byte[] decompressed = decompress(body);
                message = new Message(decompressed, properties);
            } catch (IOException e) {
                throw new MessageConversionException("解压缩失败", e);
            }
        }
        
        return delegate.fromMessage(message);
    }
    
    private byte[] compress(byte[] data) throws IOException {
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        try (GZIPOutputStream gzos = new GZIPOutputStream(baos)) {
            gzos.write(data);
        }
        return baos.toByteArray();
    }
    
    private byte[] decompress(byte[] data) throws IOException {
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        try (GZIPInputStream gzis = new GZIPInputStream(new ByteArrayInputStream(data))) {
            byte[] buffer = new byte[1024];
            int len;
            while ((len = gzis.read(buffer)) > 0) {
                baos.write(buffer, 0, len);
            }
        }
        return baos.toByteArray();
    }
}

@Component
public class VersionedMessageConverter implements MessageConverter {
    
    private final Map<String, MessageConverter> versionConverters = new HashMap<>();
    
    public VersionedMessageConverter() {
        // 注册不同版本的转换器
        versionConverters.put("1.0", createV1Converter());
        versionConverters.put("2.0", createV2Converter());
        versionConverters.put("3.0", createV3Converter());
    }
    
    @Override
    public Message toMessage(Object object, MessageProperties messageProperties) 
            throws MessageConversionException {
        
        // 获取版本
        String version = getVersion(object);
        messageProperties.setHeader("api-version", version);
        
        MessageConverter converter = versionConverters.get(version);
        if (converter == null) {
            throw new MessageConversionException("不支持的版本: " + version);
        }
        
        return converter.toMessage(object, messageProperties);
    }
    
    @Override
    public Object fromMessage(Message message) throws MessageConversionException {
        MessageProperties properties = message.getMessageProperties();
        String version = (String) properties.getHeaders().get("api-version");
        
        if (version == null) {
            version = "1.0"; // 默认版本
        }
        
        MessageConverter converter = versionConverters.get(version);
        if (converter == null) {
            throw new MessageConversionException("不支持的版本: " + version);
        }
        
        return converter.fromMessage(message);
    }
    
    private String getVersion(Object object) {
        // 根据对象类型决定版本
        if (object instanceof UserV1) {
            return "1.0";
        } else if (object instanceof UserV2) {
            return "2.0";
        } else if (object instanceof UserV3) {
            return "3.0";
        }
        return "1.0";
    }
    
    private MessageConverter createV1Converter() {
        Jackson2JsonMessageConverter converter = new Jackson2JsonMessageConverter();
        converter.getObjectMapper().configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);
        return converter;
    }
    
    private MessageConverter createV2Converter() {
        // V2 使用不同的序列化配置
        ObjectMapper mapper = new ObjectMapper();
        mapper.registerModule(new JavaTimeModule());
        return new Jackson2JsonMessageConverter(mapper);
    }
    
    private MessageConverter createV3Converter() {
        // V3 使用加密
        return new EncryptedJsonMessageConverter(new CryptoService());
    }
    
    // 不同版本的数据模型
    @Data
    public static class UserV1 {
        private Integer id;
        private String name;
    }
    
    @Data
    public static class UserV2 {
        private Integer id;
        private String username;
        private String email;
        private LocalDateTime createTime;
    }
    
    @Data
    public static class UserV3 {
        private String userId;
        private UserInfo info;
        private Map<String, Object> metadata;
        
        @Data
        public static class UserInfo {
            private String displayName;
            private String email;
            private String phone;

## 九、 常见问题

### Q1: 反序列化时类型丢失？

- 使用 `Jackson2JsonMessageConverter`并设置类型信息
    
- 在消息头中添加 `__TypeId__`
    
- 使用 `@JsonTypeInfo`注解
    

### Q2: 大消息处理？

- 使用压缩转换器
    
- 分片发送大消息
    
- 考虑使用外部存储（如OSS）存储大文件
    

### Q3: 消息格式不兼容？

- 使用 `ContentTypeDelegatingMessageConverter`
    
- 版本控制转换器
    
- 向后兼容的消息格式
    

### Q4: 性能优化？

- 复用 ObjectMapper
    
- 启用连接池
    
- 批量发送消息
    
- 使用惰性反序列化
    

消息转换器是 RabbitMQ 集成中的关键组件，合理选择和使用转换器可以简化开发、提高性能、增强系统的健壮性。
