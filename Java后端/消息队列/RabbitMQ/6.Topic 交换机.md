# RabbitMQ Topic 交换机

Topic（主题）交换机是 RabbitMQ 中最灵活、功能最强大的交换机类型，它通过 **Routing Key 的模式匹配**​ 将消息路由到队列，实现**复杂路由规则**和**消息过滤**。

## 一、 核心特性

### 1. 工作模式

```
[Topic Exchange]
              │
      ┌───────┼───────┐
      ▼       ▼       ▼
  [Queue1] [Queue2] [Queue3]
  (usa.*)  (#.news) (*.weather)
```

- **模式匹配**：使用通配符匹配 Binding Key
    
- **灵活路由**：支持多级路由键和通配符
    
- **发布/订阅增强**：比 Fanout 更精细，比 Direct 更灵活
    

### 2. 通配符规则

```
Routing Key 格式: 单词1.单词2.单词3... (用点号分隔)
               ↓
Binding Key 支持通配符:
    * (星号): 匹配一个单词
    # (井号): 匹配零个或多个单词
```

### 3. 匹配示例

```
// 绑定关系
queue1 绑定: "*.orange.*"        // 匹配: quick.orange.rabbit, lazy.orange.elephant
queue2 绑定: "*.*.rabbit"        // 匹配: quick.orange.rabbit, lazy.pink.rabbit
queue3 绑定: "lazy.#"           // 匹配: lazy.orange, lazy.orange.elephant, lazy

// 发送消息
"quick.orange.rabbit" → queue1, queue2
"lazy.orange.elephant" → queue1, queue3
"quick.orange.fox" → queue1
"lazy.brown.fox" → queue3
"lazy" → queue3
"quick" → 无匹配
```

        ConnectionFactory factory = new ConnectionFactory();
        factory.setHost("localhost");
        
        try (Connection connection = factory.newConnection();
             Channel channel = connection.createChannel()) {
            
            // 1. 声明 Topic 交换机
            channel.exchangeDeclare(EXCHANGE_NAME, "topic", true);

                String routingKey = topicMessage[0];
                String message = topicMessage[1];
                
                channel.basicPublish(EXCHANGE_NAME, routingKey, null, 
                    message.getBytes(StandardCharsets.UTF_8));
                
                System.out.printf(" [x] 发送到 '%s': %s%n", routingKey, message);
                Thread.sleep(500);
            }
            
            // 3. 发送批量消息测试
            sendBatchMessages(channel);
        }
    }

                message.getBytes(StandardCharsets.UTF_8));

        }
    }
}

import com.rabbitmq.client.*;
import java.nio.charset.StandardCharsets;
import java.util.Arrays;
import java.util.List;

public class TopicConsumer {
    
    private static final String EXCHANGE_NAME = "topic_exchange";
    private final String consumerName;
    private final List<String> bindingKeys; // 这个消费者绑定的模式
    
    public TopicConsumer(String name, String... bindingKeys) {
        this.consumerName = name;
        this.bindingKeys = Arrays.asList(bindingKeys);
    }
    
    public void start() throws Exception {
        ConnectionFactory factory = new ConnectionFactory();
        factory.setHost("localhost");
        
        Connection connection = factory.newConnection();
        Channel channel = connection.createChannel();
        
        // 1. 声明 Topic 交换机
        channel.exchangeDeclare(EXCHANGE_NAME, "topic", true);
        
        // 2. 创建队列
        String queueName = channel.queueDeclare().getQueue();
        System.out.println(consumerName + " 队列名: " + queueName);
        
        // 3. 绑定队列到交换机，使用模式匹配
        for (String bindingKey : bindingKeys) {
            channel.queueBind(queueName, EXCHANGE_NAME, bindingKey);
            System.out.println(consumerName + " 绑定模式: " + bindingKey);
        }
        
        System.out.println(consumerName + " 等待消息...\n");
        
        // 4. 创建消费者回调
        DeliverCallback deliverCallback = (consumerTag, delivery) -> {
            String message = new String(delivery.getBody(), StandardCharsets.UTF_8);
            String routingKey = delivery.getEnvelope().getRoutingKey();
            
            System.out.printf("═══════════════════════════════════%n");
            System.out.printf("%s 收到消息%n", consumerName);
            System.out.printf("路由键: %s%n", routingKey);
            System.out.printf("内容: %s%n", message);
            System.out.printf("匹配模式: %s%n", 
                findMatchingPattern(routingKey, bindingKeys));
            System.out.printf("═══════════════════════════════════%n%n");
            
            processMessage(routingKey, message);
        };
        
        // 5. 开始消费
        channel.basicConsume(queueName, true, deliverCallback, consumerTag -> {});
    }
    
    private String findMatchingPattern(String routingKey, List<String> patterns) {
        for (String pattern : patterns) {
            if (matchesPattern(routingKey, pattern)) {
                return pattern;
            }
        }
        return "未知";
    }
    
    private boolean matchesPattern(String routingKey, String pattern) {
        // 简化的模式匹配逻辑
        String[] routingParts = routingKey.split("\\.");
        String[] patternParts = pattern.split("\\.");
        
        int rIdx = 0, pIdx = 0;
        
        while (rIdx < routingParts.length && pIdx < patternParts.length) {
            if (patternParts[pIdx].equals("#")) {
                return true; // # 匹配剩余所有
            } else if (patternParts[pIdx].equals("*") || 
                      patternParts[pIdx].equals(routingParts[rIdx])) {
                rIdx++;
                pIdx++;
            } else {
                return false;
            }
        }
        
        return rIdx == routingParts.length && 
               (pIdx == patternParts.length || patternParts[pIdx].equals("#"));
    }
    
    private void processMessage(String routingKey, String message) throws InterruptedException {
        // 模拟不同主题的处理逻辑
        if (routingKey.startsWith("stock.")) {
            System.out.println(consumerName + " 处理股票数据...");
            Thread.sleep(1000);
        } else if (routingKey.startsWith("news.")) {
            System.out.println(consumerName + " 处理新闻...");
            Thread.sleep(800);
        } else if (routingKey.startsWith("weather.")) {
            System.out.println(consumerName + " 处理天气数据...");
            Thread.sleep(600);
        } else if (routingKey.startsWith("order.")) {
            System.out.println(consumerName + " 处理订单事件...");
            Thread.sleep(1200);
        } else if (routingKey.startsWith("user.")) {
            System.out.println(consumerName + " 处理用户事件...");
            Thread.sleep(700);
        } else if (routingKey.startsWith("system.")) {
            System.out.println(consumerName + " 处理系统事件...");
            Thread.sleep(1500);
        }
    }
    
    public static void main(String[] args) throws Exception {
        System.out.println("启动多个主题消费者...\n");
        
        // 启动不同模式的消费者
        Thread stockConsumer = new Thread(() -> {
            try {
                // 消费所有股票相关消息
                new TopicConsumer("股票监控器", "stock.*").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread usStockConsumer = new Thread(() -> {
            try {
                // 只消费美国股票
                new TopicConsumer("美股监控器", "stock.us.*").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread newsConsumer = new Thread(() -> {
            try {
                // 消费所有新闻
                new TopicConsumer("新闻订阅器", "news.#").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread techNewsConsumer = new Thread(() -> {
            try {
                // 只消费科技新闻
                new TopicConsumer("科技新闻", "news.tech.*").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread weatherConsumer = new Thread(() -> {
            try {
                // 消费所有天气消息
                new TopicConsumer("天气服务", "weather.*").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread orderConsumer = new Thread(() -> {
            try {
                // 消费所有订单事件
                new TopicConsumer("订单处理器", "order.*").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread systemConsumer = new Thread(() -> {
            try {
                // 消费系统错误和警告
                new TopicConsumer("系统监控", "system.error", "system.warning").start();
            } catch (Exception e) {
                e.printStackTrace();
            }
        });
        
        Thread allConsumer = new Thread(() -> {
            try {
                // 消费所有消息
                new TopicConsumer("全量订阅器", "#").start();
            } catch (Exception e) {
                e.printStackTrace();

        });
        
        // 启动所有消费者

        techNewsConsumer.start();
        weatherConsumer.start();
        orderConsumer.start();

@Component
public class TopicProducer {
    
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    @Autowired
    private TopicExchange topicExchange;
    
    /**
     * 发送股票消息
     */
    public void sendStockMessage(String market, String symbol, String data) {
        String routingKey = String.format("stock.%s.%s", market.toLowerCase(), symbol.toLowerCase());
        String message = String.format("%s:%s - %s", market.toUpperCase(), symbol.toUpperCase(), data);
        
        sendMessage(routingKey, message, "stock");
    }
    
    /**
     * 发送新闻消息
     */
    public void sendNewsMessage(String category, String subCategory, String title) {
        String routingKey = String.format("news.%s.%s", category.toLowerCase(), subCategory.toLowerCase());
        String message = String.format("[%s] %s", category.toUpperCase(), title);
        
        sendMessage(routingKey, message, "news");
    }
    
    /**
     * 发送订单事件
     */
    public void sendOrderEvent(String orderId, String event) {
        String routingKey = "order." + event.toLowerCase();
        OrderEvent orderEvent = new OrderEvent(orderId, event, new Date());
        
        rabbitTemplate.convertAndSend(topicExchange.getName(), routingKey, orderEvent);
        System.out.println("发送订单事件: " + routingKey);
    }
    
    /**
     * 发送用户活动
     */
    public void sendUserActivity(String userId, String action) {
        String routingKey = "user." + action.toLowerCase();
        UserActivity activity = new UserActivity(userId, action, new Date());
        
        rabbitTemplate.convertAndSend(topicExchange.getName(), routingKey, activity);
        System.out.println("发送用户活动: " + routingKey);
    }
    
    /**
     * 发送系统日志
     */
    public void sendSystemLog(String level, String component, String message) {
        String routingKey = String.format("system.%s.%s", level.toLowerCase(), component.toLowerCase());
        SystemLog log = new SystemLog(level, component, message, new Date());
        
        rabbitTemplate.convertAndSend(topicExchange.getName(), routingKey, log);
        System.out.println("发送系统日志: " + routingKey);
    }
    
    /**
     * 发送气象数据
     */
    public void sendWeatherData(String city, String data) {
        String routingKey = "weather." + city.toLowerCase().replace(" ", ".");
        rabbitTemplate.convertAndSend(topicExchange.getName(), routingKey, data);
        System.out.println("发送天气数据: " + routingKey);
    }
    
    /**
     * 发送测试消息
     */
    public void sendTestMessages() {
        // 股票
        sendStockMessage("us", "AAPL", "价格上涨 2.5%");
        sendStockMessage("us", "GOOGL", "价格下跌 1.2%");
        sendStockMessage("cn", "SH", "上证指数 3200点");
        sendStockMessage("cn", "SZ", "深证成指 11000点");
        
        // 新闻
        sendNewsMessage("world", "politics", "联合国大会召开");
        sendNewsMessage("world", "sports", "世界杯决赛");
        sendNewsMessage("tech", "ai", "AI 取得新突破");
        sendNewsMessage("tech", "blockchain", "区块链新应用");
        sendNewsMessage("business", "finance", "央行降息");
        
        // 订单
        sendOrderEvent("ORD001", "created");
        sendOrderEvent("ORD001", "paid");
        sendOrderEvent("ORD001", "shipped");
        sendOrderEvent("ORD002", "cancelled");
        
        // 用户
        sendUserActivity("user001", "registered");
        sendUserActivity("user001", "login");
        sendUserActivity("user001", "logout");
        sendUserActivity("user001", "profile_update");
        
        // 系统

        sendSystemLog("warning", "disk", "空间不足");

        sendWeatherData("beijing", "多云, 30°C");
        sendWeatherData("shanghai", "雨, 28°C");

    private void sendMessage(String routingKey, String message, String type) {
        Map<String, Object> headers = new HashMap<>();
        headers.put("message-type", type);
        headers.put("timestamp", System.currentTimeMillis());
        
        MessageProperties props = MessagePropertiesBuilder.newInstance()
                .setContentType(MessageProperties.CONTENT_TYPE_TEXT_PLAIN)
                .setDeliveryMode(MessageDeliveryMode.PERSISTENT)
                .setHeaders(headers)
                .build();
        
        Message msg = MessageBuilder.withBody(message.getBytes())
                .andProperties(props)
                .build();
        
        rabbitTemplate.send(topicExchange.getName(), routingKey, msg);
        System.out.println("发送消息 [" + type + "]: " + routingKey);
    }
    
    // 数据模型

    @NoArgsConstructor

@Service
public class DynamicTopicService {
    
    @Autowired
    private AmqpAdmin amqpAdmin;
    
    @Autowired
    private TopicExchange topicExchange;
    
    /**
     * 动态订阅主题
     */
    public String subscribeToTopic(String queueName, String topicPattern) {
        // 创建队列
        Queue queue = QueueBuilder.durable(queueName)
                .withArgument("x-queue-mode", "lazy")
                .build();
        
        amqpAdmin.declareQueue(queue);
        
        // 绑定到主题
        Binding binding = new Binding(
            queueName,
            Binding.DestinationType.QUEUE,
            topicExchange.getName(),
            topicPattern,
            null
        );
        
        amqpAdmin.declareBinding(binding);
        
        System.out.println("队列 " + queueName + " 已订阅主题: " + topicPattern);
        return queueName;
    }
    
    /**
     * 取消订阅
     */
    public void unsubscribeFromTopic(String queueName, String topicPattern) {
        Binding binding = new Binding(
            queueName,
            Binding.DestinationType.QUEUE,
            topicExchange.getName(),
            topicPattern,
            null
        );
        
        amqpAdmin.removeBinding(binding);
        System.out.println("队列 " + queueName + " 已取消订阅: " + topicPattern);
    }
    
    /**
     * 获取主题的所有订阅者
     */
    public List<String> getTopicSubscribers(String topicPattern) {
        return rabbitTemplate.execute(channel -> {
            List<String> subscribers = new ArrayList<>();
            
            try {
                AMQP.Queue.BindOk[] bindings = channel.queueBindings(topicExchange.getName());
                
                for (AMQP.Queue.BindOk binding : bindings) {
                    if (topicMatches(binding.getRoutingKey(), topicPattern)) {
                        subscribers.add(binding.getQueue());
                    }
                }
            } catch (Exception e) {
                e.printStackTrace();
            }
            
            return subscribers;
        });
    }
    
    /**
     * 创建临时订阅
     */
    public TemporarySubscription createTemporarySubscription(String topicPattern) {
        // 创建临时队列
        Queue queue = new Queue("", false, true, true);
        String queueName = amqpAdmin.declareQueue(queue);
        
        // 绑定
        Binding binding = BindingBuilder.bind(queue)
                .to(topicExchange)
                .with(topicPattern);
        amqpAdmin.declareBinding(binding);
        
        return new TemporarySubscription(queueName, topicPattern);
    }
    
    private boolean topicMatches(String routingKey, String pattern) {
        // 主题模式匹配
        return routingKey.matches(pattern.replace(".", "\\.")
                                        .replace("*", "[^.]+")
                                        .replace("#", ".+"));
    }
    
    @Data
    @AllArgsConstructor
    public static class TemporarySubscription {
        private String queueName;
        private String topicPattern;
    }
}

@Service
public class TopicRoutingManager {
    
    private final Map<String, TopicRoute> topicRoutes = new ConcurrentHashMap<>();
    
    @Autowired
    private DynamicTopicService dynamicTopicService;
    
    /**
     * 注册主题路由
     */
    public void registerTopicRoute(String serviceName, String topicPattern, String handler) {
        String queueName = "route." + serviceName + "." + UUID.randomUUID().toString().substring(0, 8);
        
        // 订阅主题
        dynamicTopicService.subscribeToTopic(queueName, topicPattern);
        
        // 保存路由信息
        TopicRoute route = new TopicRoute();
        route.setServiceName(serviceName);
        route.setTopicPattern(topicPattern);
        route.setQueueName(queueName);
        route.setHandler(handler);
        route.setCreatedAt(new Date());
        
        topicRoutes.put(queueName, route);
        
        System.out.println("注册主题路由: " + serviceName + " -> " + topicPattern);
    }
    
    /**
     * 根据内容动态路由
     */
    public void dynamicRouteByContent(String message, Map<String, Object> metadata) {
        String topic = determineTopic(message, metadata);
        
        if (topic != null) {
            sendToTopic(topic, message, metadata);
        }
    }
    
    /**
     * 多级主题路由
     */
    public void multiLevelRoute(String baseTopic, String message, Map<String, String> attributes) {
        StringBuilder topicBuilder = new StringBuilder(baseTopic);
        
        for (Map.Entry<String, String> entry : attributes.entrySet()) {
            if (entry.getValue() != null && !entry.getValue().isEmpty()) {
                topicBuilder.append(".").append(entry.getKey()).append(".").append(entry.getValue());
            }
        }
        
        String fullTopic = topicBuilder.toString();
        sendToTopic(fullTopic, message, Map.of("attributes", attributes));
    }
    
    private String determineTopic(String message, Map<String, Object> metadata) {
        // 根据消息内容和元数据决定主题
        String messageType = (String) metadata.get("type");
        
        if ("STOCK".equals(messageType)) {
            String market = (String) metadata.get("market");
            String symbol = (String) metadata.get("symbol");
            return "stock." + market + "." + symbol;
        } else if ("NEWS".equals(messageType)) {
            String category = (String) metadata.get("category");
            String subCategory = (String) metadata.get("subCategory");
            return "news." + category + "." + subCategory;
        } else if ("ORDER".equals(messageType)) {
            String action = (String) metadata.get("action");
            return "order." + action;
        }
        
        return "default";
    }
    
    private void sendToTopic(String topic, String message, Map<String, Object> metadata) {
        // 实际发送逻辑
        System.out.println("发送到主题: " + topic);
        System.out.println("消息: " + message);
        System.out.println("元数据: " + metadata);
    }
    
    @Data
    public static class TopicRoute {
        private String serviceName;
        private String topicPattern;
        private String queueName;
        private String handler;
        private Date createdAt;
    }
}

## 七、 最佳实践

1. **命名规范**：
    
    - 使用有意义的层次结构：`domain.entity.action`
        
    - 保持主题简洁但具有描述性
        
    - 避免过深的层次（一般3-4级）
        
    
2. **通配符使用**：
    
    - `*`用于匹配一个单词
        
    - `#`用于匹配多个单词
        
    - 谨慎使用 `#`，可能导致意外的消息接收
        
    
3. **性能优化**：
    
    - 主题匹配比直接路由稍慢
        
    - 避免过于复杂的模式
        
    - 监控主题匹配的性能
        
    
4. **错误处理**：
    
    - 为未匹配的消息设置默认处理器
        
    - 监控未路由的消息
        
    - 实现死信队列
        
    
5. **监控调试**：
    
    - 记录消息的路由路径
        
    - 监控主题订阅情况
        
    - 实现消息追踪
        
    

## 八、 常见问题

### Q1: Topic 和 Direct 的区别？

- **Topic**：支持通配符模式匹配
    
- **Direct**：需要精确匹配
    

### Q2: 主题层次应该多深？

- 一般3-4级比较合适
    
- 太深难以管理
    
- 太浅无法提供足够的灵活性
    

### Q3: 如何避免消息被多个消费者接收？

- 使用独占队列
    
- 确保主题模式不重叠
    
- 消费者端去重
    

### Q4: 主题模式性能如何？

- 主题匹配比直接路由稍慢
    
- 对于高性能场景，考虑使用 Direct
    
- 大多数场景下性能差异可接受
    

Topic 交换机是 RabbitMQ 中最强大的路由机制，特别适合需要灵活消息过滤和复杂路由规则的场景。通过合理的主题设计，可以构建出既灵活又易于维护的消息系统。
