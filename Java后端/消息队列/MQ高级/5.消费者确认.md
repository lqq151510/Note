# RabbitMQ 消费者确认机制（Consumer Acknowledgements）

消费者确认机制是保证消息**可靠消费**的核心机制，确保消息被消费者**正确处理**后才从队列中移除。

## 一、 三种确认模式

### 1. 自动确认（AUTO，不推荐）

```
@Configuration
public class AutoAckConfig {
    
    /**
     * 自动确认模式
     * 消息发送给消费者后立即从队列删除
     * ❌ 风险：消息可能处理失败但已被删除
     */
    @Bean
    public SimpleRabbitListenerContainerFactory autoAckContainerFactory(
            ConnectionFactory connectionFactory) {
        
        SimpleRabbitListenerContainerFactory factory = new SimpleRabbitListenerContainerFactory();
        factory.setConnectionFactory(connectionFactory);
        
        // 设置自动确认
        factory.setAcknowledgeMode(AcknowledgeMode.AUTO);
        
        return factory;
    }
    
    /**
     * 自动确认消费者
     */
    @RabbitListener(queues = "auto.ack.queue", 
                   containerFactory = "autoAckContainerFactory")
    public void handleAutoAck(String message) {
        System.out.println("自动确认模式处理: " + message);
        
        // 如果这里抛出异常，消息已丢失
        if (message.contains("error")) {
            throw new RuntimeException("模拟处理失败");
        }
    }
}
```

### 2. 手动确认（MANUAL，推荐）

```
@Configuration
public class ManualAckConfig {
    
    /**
     * 手动确认模式
     * ✅ 安全：消息处理成功后才确认删除
     */
    @Bean
    public SimpleRabbitListenerContainerFactory manualAckContainerFactory(
            ConnectionFactory connectionFactory) {
        
        SimpleRabbitListenerContainerFactory factory = new SimpleRabbitListenerContainerFactory();
        factory.setConnectionFactory(connectionFactory);
        
        // 设置手动确认
        factory.setAcknowledgeMode(AcknowledgeMode.MANUAL);
        
        // 设置预取数量
        factory.setPrefetchCount(1); // 每次只给消费者一条消息
        
        return factory;
    }
    
    /**
     * 手动确认消费者 - 基本模式
     */
    @RabbitListener(queues = "manual.ack.queue", 
                   containerFactory = "manualAckContainerFactory")
    public void handleManualAck(String message,
                               Channel channel,
                               @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag) 
            throws IOException {
        
        System.out.println("手动确认处理消息: " + message);
        
        try {
            // 业务处理逻辑
            processMessage(message);
            
            // ✅ 处理成功，确认消息
            // 参数1: deliveryTag
            // 参数2: multiple 是否批量确认
            channel.basicAck(deliveryTag, false);
            
            System.out.println("消息确认成功: " + deliveryTag);
            
        } catch (Exception e) {
            System.err.println("消息处理失败: " + e.getMessage());
            
            // ❌ 处理失败，拒绝消息
            // 参数1: deliveryTag
            // 参数2: multiple 是否批量拒绝
            // 参数3: requeue 是否重新入队
            channel.basicNack(deliveryTag, false, true);
            
            System.err.println("消息重新入队: " + deliveryTag);
        }
    }
    
    /**
     * 手动确认消费者 - 带重试控制
     */
    @RabbitListener(queues = "retry.control.queue")
    public void handleWithRetryControl(String message,
                                      Channel channel,
                                      @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag,
                                      @Header(name = "x-death", required = false) 
                                      List<Map<String, Object>> deathRecords) 
            throws IOException {
        
        System.out.println("处理消息: " + message);
        
        // 检查重试次数
        int retryCount = getRetryCount(deathRecords);
        
        if (retryCount >= 3) {
            // 达到最大重试次数
            System.err.println("达到最大重试次数，消息: " + message);
            
            // 发送到死信队列
            channel.basicNack(deliveryTag, false, false);
            
            // 或者可以记录到数据库
            saveToFailedMessages(message, retryCount);
            
        } else {
            try {
                processMessage(message);
                channel.basicAck(deliveryTag, false);
                
            } catch (Exception e) {
                // 重新入队等待重试
                channel.basicNack(deliveryTag, false, true);
                System.err.println("处理失败，第" + (retryCount + 1) + "次重试");
            }
        }
    }
    
    private int getRetryCount(List<Map<String, Object>> deathRecords) {
        if (deathRecords == null || deathRecords.isEmpty()) {
            return 0;
        }
        
        int count = 0;
        for (Map<String, Object> record : deathRecords) {
            if (record.containsKey("count")) {
                count += (Integer) record.get("count");
            }
        }
        return count;
    }
    
    private void processMessage(String message) throws Exception {
        // 模拟业务处理
        if (message.contains("fail")) {
            throw new Exception("模拟处理失败");
        }
        
        // 处理逻辑
        Thread.sleep(100);
    }
    
    private void saveToFailedMessages(String message, int retryCount) {
        // 保存失败消息
        System.out.println("保存失败消息: " + message + ", 重试次数: " + retryCount);
    }
}
```

```
     * 条件确认消费者
     */
    @RabbitListener(queues = "conditional.ack.queue", 
                   containerFactory = "conditionalAckContainerFactory")

        System.out.println("条件确认处理: " + message);
        
        try {
            processMessage(message);

            System.err.println("处理失败: " + e.getMessage());
            
            // 抛出 AmqpRejectAndDontRequeueException 不重新入队
            throw new AmqpRejectAndDontRequeueException(e);
            
            // 或者抛出 ImmediateRequeueAmqpException 立即重新入队
            // throw new ImmediateRequeueAmqpException(e);
        }
    }
}
```

## 二、 高级确认机制

### 1. 批量确认

```
@Component
public class BatchAckHandler {
    
    /**
     * 批量消费者
     */
    @RabbitListener(queues = "batch.ack.queue", 
                   containerFactory = "batchContainerFactory")
    public void handleBatchAck(List<Message> messages, Channel channel) 
            throws IOException {
        
        System.out.println("批量处理 " + messages.size() + " 条消息");
        
        List<Long> successTags = new ArrayList<>();
        List<Long> failureTags = new ArrayList<>();
        
        for (Message message : messages) {
            long deliveryTag = message.getMessageProperties().getDeliveryTag();
            String body = new String(message.getBody());
            
            try {
                // 处理消息
                processMessage(body);
                successTags.add(deliveryTag);
                
            } catch (Exception e) {
                System.err.println("处理失败: " + e.getMessage());
                failureTags.add(deliveryTag);
            }
        }
        
        // ✅ 批量确认成功消息
        if (!successTags.isEmpty()) {
            long lastDeliveryTag = successTags.get(successTags.size() - 1);
            channel.basicAck(lastDeliveryTag, true); // multiple=true
            System.out.println("批量确认 " + successTags.size() + " 条成功消息");
        }
        
        // ❌ 批量拒绝失败消息
        if (!failureTags.isEmpty()) {
            long lastDeliveryTag = failureTags.get(failureTags.size() - 1);
            channel.basicNack(lastDeliveryTag, true, true); // 重新入队
            System.out.println("批量拒绝 " + failureTags.size() + " 条失败消息");
        }
    }
    
    /**
     * 部分确认
     */
    public void handlePartialAck(List<Message> messages, Channel channel) 
            throws IOException {
        
        Map<Boolean, List<Long>> resultMap = new HashMap<>();
        resultMap.put(true, new ArrayList<>());  // 成功
        resultMap.put(false, new ArrayList<>()); // 失败
        
        for (Message message : messages) {
            long deliveryTag = message.getMessageProperties().getDeliveryTag();
            
            try {
                processMessage(new String(message.getBody()));
                resultMap.get(true).add(deliveryTag);
            } catch (Exception e) {
                resultMap.get(false).add(deliveryTag);
            }
        }
        
        // 确认成功消息
        for (Long tag : resultMap.get(true)) {
            channel.basicAck(tag, false);
        }
        
        // 拒绝失败消息
        for (Long tag : resultMap.get(false)) {
            channel.basicNack(tag, false, true);
        }
    }
}
```

### 2. 延迟确认

```
@Component
public class DelayedAckHandler {
    
    private final Map<Long, MessageContext> pendingAcks = new ConcurrentHashMap<>();
    private final ScheduledExecutorService scheduler = Executors.newScheduledThreadPool(1);
    
    /**
     * 接收消息但不立即确认
     */
    @RabbitListener(queues = "delayed.ack.queue")
    public void receiveAndDelayAck(String message,
                                  Channel channel,
                                  @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag) {
        
        System.out.println("接收消息: " + message);
        
        // 保存消息上下文
        MessageContext context = new MessageContext(message, channel, deliveryTag);
        pendingAcks.put(deliveryTag, context);
        
        // 异步处理
        processAsync(context);
    }
    
    /**
     * 异步处理
     */
    private void processAsync(MessageContext context) {
        CompletableFuture.runAsync(() -> {
            try {
                // 模拟长时间处理
                Thread.sleep(5000);
                
                // 处理逻辑
                processMessage(context.getMessage());
                
                // 处理成功，确认消息
                context.getChannel().basicAck(context.getDeliveryTag(), false);
                System.out.println("延迟确认成功: " + context.getDeliveryTag());
                
            } catch (Exception e) {
                System.err.println("异步处理失败: " + e.getMessage());
                
                try {
                    // 处理失败，拒绝消息
                    context.getChannel().basicNack(context.getDeliveryTag(), false, true);
                } catch (IOException ioException) {
                    ioException.printStackTrace();
                }
                
            } finally {
                // 清理
                pendingAcks.remove(context.getDeliveryTag());
            }
        });
    }
    
    /**
     * 定时确认检查
     */
    @Scheduled(fixedDelay = 30000) // 每30秒检查
    public void checkPendingAcks() {
        long now = System.currentTimeMillis();
        
        pendingAcks.values().removeIf(context -> {
            if (now - context.getReceiveTime() > 300000) { // 超过5分钟
                try {
                    System.err.println("消息处理超时: " + context.getMessage());
                    context.getChannel().basicNack(context.getDeliveryTag(), false, true);
                    return true;
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
            return false;
        });
    }
    
    /**
     * 消息上下文
     */
    @Data
    @AllArgsConstructor
    private static class MessageContext {
        private String message;
        private Channel channel;
        private long deliveryTag;
        private long receiveTime = System.currentTimeMillis();
    }
}
```

## 三、 确认与重试策略

### 1. 智能重试策略

```
@Component
public class SmartRetryStrategy {
    
    private final Map<String, RetryStats> retryStats = new ConcurrentHashMap<>();
    
    /**
     * 智能确认处理
     */
    @RabbitListener(queues = "smart.retry.queue", 
                   ackMode = "MANUAL")
    public void handleWithSmartRetry(String message,
                                    Channel channel,
                                    @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag,
                                    @Header(AmqpHeaders.CONSUMER_QUEUE) String queue) 
            throws IOException {
        
        RetryStats stats = retryStats.computeIfAbsent(queue, k -> new RetryStats());
        String messageId = extractMessageId(message);
        
        try {
            // 检查重试次数
            int retryCount = stats.getRetryCount(messageId);
            
            if (retryCount >= getMaxRetries(message)) {
                // 达到最大重试次数
                handleMaxRetries(message, messageId, stats, channel, deliveryTag);
                return;
            }
            
            // 处理消息
            processWithRetry(message, retryCount);
            
            // 处理成功
            stats.recordSuccess(messageId);
            channel.basicAck(deliveryTag, false);
            
        } catch (Exception e) {
            // 处理失败
            stats.recordFailure(messageId);
            
            // 智能决策：是否重新入队
            boolean shouldRequeue = shouldRequeue(message, e, stats.getRetryCount(messageId));
            
            if (shouldRequeue) {
                // 重新入队
                channel.basicNack(deliveryTag, false, true);
                
                // 记录重试
                stats.incrementRetry(messageId);
                
                // 计算延迟
                long delay = calculateRetryDelay(stats.getRetryCount(messageId));
                System.out.println("第" + stats.getRetryCount(messageId) + "次重试，延迟" + delay + "ms");
                
            } else {
                // 发送到死信队列
                channel.basicNack(deliveryTag, false, false);
                stats.remove(messageId);
                
                // 记录失败
                saveToDeadLetter(message, e);
            }
        }
    }
    
    /**
     * 智能判断是否重新入队
     */
    private boolean shouldRequeue(String message, Exception e, int retryCount) {
        // 1. 检查异常类型
        if (e instanceof IllegalArgumentException) {
            return false; // 参数错误，不重试
        }
        
        if (e instanceof IOException) {
            return true; // 网络异常，重试
        }
        
        // 2. 检查重试次数
        if (retryCount >= 5) {
            return false; // 超过最大重试次数
        }
        
        // 3. 检查消息内容
        if (message.contains("no-retry")) {
            return false; // 特定消息不重试
        }
        
        // 4. 检查业务规则
        return isRecoverableError(e);
    }
    
    /**
     * 计算重试延迟
     */
    private long calculateRetryDelay(int retryCount) {
        // 指数退避 + 随机抖动
        long baseDelay = 1000L * (1L << Math.min(retryCount, 10));
        long jitter = (long) (Math.random() * 1000);
        return Math.min(baseDelay + jitter, 30000L);
    }
    
    private int getMaxRetries(String message) {
        // 根据消息内容决定最大重试次数
        if (message.contains("urgent")) {
            return 3;
        } else if (message.contains("normal")) {
            return 5;
        } else {
            return 10;
        }
    }
    
    private void handleMaxRetries(String message, String messageId, 
                                 RetryStats stats, Channel channel, 
                                 long deliveryTag) throws IOException {
        
        System.err.println("达到最大重试次数: " + message);
        
        // 发送到死信队列
        channel.basicNack(deliveryTag, false, false);
        
        // 清理统计
        stats.remove(messageId);
        
        // 记录失败
        saveToMaxRetries(message);
    }
    
    private String extractMessageId(String message) {
        // 从消息中提取ID
        return Integer.toHexString(message.hashCode());
    }
    
    private boolean isRecoverableError(Exception e) {
        // 判断是否可恢复错误
        return !(e instanceof IllegalArgumentException);
    }
    
    private void processWithRetry(String message, int retryCount) throws Exception {
        // 带重试的处理逻辑
        if (retryCount > 0) {
            // 如果是重试，可以做一些特殊处理
            System.out.println("第" + (retryCount + 1) + "次尝试处理");
        }
        
        processMessage(message);
    }
    
    private void saveToDeadLetter(String message, Exception e) {
        // 保存到死信记录
        System.err.println("发送到死信队列: " + message);
    }
    
    private void saveToMaxRetries(String message) {
        // 保存达到最大重试的消息
        System.err.println("记录最大重试失败: " + message);
    }
    
    /**
     * 重试统计
     */
    @Data
    public static class RetryStats {
        private Map<String, Integer> messageRetries = new ConcurrentHashMap<>();
        private Map<String, Long> lastRetryTime = new ConcurrentHashMap<>();
        
        public int getRetryCount(String messageId) {
            return messageRetries.getOrDefault(messageId, 0);
        }
        
        public void recordSuccess(String messageId) {
            messageRetries.remove(messageId);
            lastRetryTime.remove(messageId);
        }
        
        public void recordFailure(String messageId) {
            // 记录失败，但不增加重试计数
            lastRetryTime.put(messageId, System.currentTimeMillis());
        }
        
        public void incrementRetry(String messageId) {
            int count = getRetryCount(messageId);
            messageRetries.put(messageId, count + 1);
            lastRetryTime.put(messageId, System.currentTimeMillis());
        }
        
        public void remove(String messageId) {
            messageRetries.remove(messageId);
            lastRetryTime.remove(messageId);
        }
    }
}
```

### 2. Spring Retry 集成

```
@Configuration
@EnableRetry
public class SpringRetryIntegration {
    
    /**
     * 配置重试容器工厂
     */
    @Bean
    public SimpleRabbitListenerContainerFactory retryContainerFactory(
            ConnectionFactory connectionFactory) {
        
        SimpleRabbitListenerContainerFactory factory = new SimpleRabbitListenerContainerFactory();
        factory.setConnectionFactory(connectionFactory);
        
        // 设置手动确认
        factory.setAcknowledgeMode(AcknowledgeMode.MANUAL);
        
        // 配置重试
        factory.setAdviceChain(retryInterceptor());
        
        return factory;
    }
    
    /**
     * 重试拦截器
     */
    @Bean
    public Advice retryInterceptor() {
        return RetryInterceptorBuilder.stateless()
            .maxAttempts(3)  // 最大尝试次数
            .backOffOptions(1000, 2.0, 10000) // 初始1秒，倍数2，最大10秒
            .recoverer(new RejectAndDontRequeueRecoverer()) // 重试耗尽后的处理
            .build();
    }
    
    /**
     * 使用 @Retryable 注解
     */
    @Service
    public class RetryableConsumer {
        
        @RabbitListener(queues = "retryable.queue", 
                       containerFactory = "retryContainerFactory")
        @Retryable(
            value = {Exception.class},
            maxAttempts = 3,
            backoff = @Backoff(delay = 1000, multiplier = 2.0)
        )
        public void handleWithAnnotation(String message,
                                        Channel channel,
                                        @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag) 
                throws IOException {
            
            System.out.println("处理消息: " + message);
            
            try {
                processMessage(message);
                channel.basicAck(deliveryTag, false);
                
            } catch (Exception e) {
                System.err.println("处理失败，触发重试: " + e.getMessage());
                throw e; // 抛出异常触发重试
            }
        }
        
        /**
         * 恢复方法
         */
        @Recover
        public void recover(Exception e, String message, 
                          Channel channel, long deliveryTag) throws IOException {
            
            System.err.println("重试耗尽，发送到死信队列: " + message);
            
            // 拒绝消息，不重新入队
            channel.basicNack(deliveryTag, false, false);
            
            // 记录失败
            saveFailedMessage(message, e);
        }
        
        private void saveFailedMessage(String message, Exception e) {
            // 保存失败消息
            System.err.println("保存失败消息: " + message);
        }
    }
}
```

## 四、 确认与死信队列

### 1. 死信队列配置

```
@Configuration
public class DeadLetterQueueConfig {
    
    /**
     * 配置死信队列
     */
    @Bean
    public Queue mainQueue() {
        Map<String, Object> args = new HashMap<>();
        
        // 配置死信交换机
        args.put("x-dead-letter-exchange", "dlx.exchange");
        args.put("x-dead-letter-routing-key", "dlx.routing.key");
        
        // 可选：设置消息TTL
        args.put("x-message-ttl", 30000); // 30秒
        
        return QueueBuilder.durable("main.queue")
                .withArguments(args)
                .build();
    }
    
    /**
     * 死信交换机
     */
    @Bean
    public DirectExchange dlxExchange() {
        return new DirectExchange("dlx.exchange", true, false);
    }
    
    /**
     * 死信队列
     */
    @Bean
    public Queue dlxQueue() {
        return QueueBuilder.durable("dlx.queue")
                .build();
    }
    
    /**
     * 绑定死信队列
     */
    @Bean
    public Binding dlxBinding() {
        return BindingBuilder.bind(dlxQueue())
                .to(dlxExchange())
                .with("dlx.routing.key");
    }
    
    /**
     * 主队列消费者
     */
    @Component
    public class MainQueueConsumer {
        
        @RabbitListener(queues = "main.queue")
        public void handleMainQueue(String message,
                                   Channel channel,
                                   @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag) 
                throws IOException {
            
            System.out.println("处理主队列消息: " + message);
            
            try {
                processMessage(message);
                channel.basicAck(deliveryTag, false);
                
            } catch (Exception e) {
                System.err.println("处理失败，发送到死信队列: " + e.getMessage());
                
                // 拒绝消息，不重新入队，触发死信
                channel.basicNack(deliveryTag, false, false);
            }
        }
    }
    
    /**
     * 死信队列消费者
     */
    @Component
    public class DeadLetterConsumer {
        
        @RabbitListener(queues = "dlx.queue")
        public void handleDeadLetter(String message,
                                    @Header(AmqpHeaders.RECEIVED_ROUTING_KEY) String routingKey,
                                    @Header(AmqpHeaders.RECEIVED_EXCHANGE) String exchange,
                                    @Header(name = "x-death", required = false) 
                                    List<Map<String, Object>> deathRecords) {
            
            System.err.println("收到死信消息:");
            System.err.println("原始消息: " + message);
            System.err.println("来源交换: " + exchange);
            System.err.println("来源路由键: " + routingKey);
            
            if (deathRecords != null) {
                for (Map<String, Object> record : deathRecords) {
                    System.err.println("死亡记录: " + record);
                }
            }
            
            // 处理死信
            handleDeadLetterMessage(message, deathRecords);
        }
        
        private void handleDeadLetterMessage(String message,
```

```
@Component
public class AckMonitor {
    
    private final Counter ackCounter;
    private final Counter nackCounter;
    private final Counter rejectCounter;
    private final Timer ackTimer;
    
    private final Map<String, QueueMetrics> queueMetrics = new ConcurrentHashMap<>();
    
    public AckMonitor(MeterRegistry meterRegistry) {
        // 注册指标
        this.ackCounter = Counter.builder("rabbitmq.consumer.ack")
            .description("消息确认成功次数")
            .register(meterRegistry);
        
        this.nackCounter = Counter.builder("rabbitmq.consumer.nack")
            .description("消息确认失败次数")
            .register(meterRegistry);
        
        this.rejectCounter = Counter.builder("rabbitmq.consumer.reject")
            .description("消息拒绝次数")
            .register(meterRegistry);
        
        this.ackTimer = Timer.builder("rabbitmq.consumer.processing.time")
            .description("消息处理耗时")
            .register(meterRegistry);
    }
    
    /**
     * 监控确认
     */
    public <T> T monitorAck(Supplier<T> operation, String queue, String messageId) {
        long startTime = System.currentTimeMillis();
        
        try {
            T result = operation.get();
            
            // 记录成功
            ackCounter.increment();
            recordAckSuccess(queue, System.currentTimeMillis() - startTime);
            
            return result;
            
        } catch (Exception e) {
            // 记录失败
            nackCounter.increment();
            recordAckFailure(queue, e);
            
            throw e;
        }
    }
    
    /**
     * 记录确认成功
     */
    public void recordAckSuccess(String queue, long duration) {
        QueueMetrics metrics = queueMetrics.computeIfAbsent(queue, k -> new QueueMetrics());
        metrics.recordSuccess(duration);
        
        // 记录到计时器
        ackTimer.record(duration, TimeUnit.MILLISECONDS);
        
        System.out.println("确认成功 - 队列: " + queue + ", 耗时: " + duration + "ms");
    }
    
    /**
     * 记录确认失败
     */
    public void recordAckFailure(String queue, Exception e) {
        QueueMetrics metrics = queueMetrics.computeIfAbsent(queue, k -> new QueueMetrics());
        metrics.recordFailure(e);
        
        System.err.println("确认失败 - 队列: " + queue + ", 错误: " + e.getMessage());
    }
    
    /**
     * 获取队列统计
     */
    public QueueStats getQueueStats(String queue) {
        QueueMetrics metrics = queueMetrics.get(queue);
        if (metrics != null) {
            return metrics.getStats();
        }
        return new QueueStats();
    }
    
    /**
     * 健康检查
     */
    public Health ackHealthCheck() {
        Map<String, Object> details = new HashMap<>();
        
        double overallSuccessRate = queueMetrics.values().stream()
            .mapToDouble(m -> m.getStats().getSuccessRate())
            .average()
            .orElse(1.0);
        
        details.put("overallSuccessRate", overallSuccessRate);
        details.put("totalAck", ackCounter.count());
        details.put("totalNack", nackCounter.count());
        
        queueMetrics.forEach((queue, metrics) -> {
            details.put(queue + ".successRate", metrics.getStats().getSuccessRate());
            details.put(queue + ".avgDuration", metrics.getStats().getAvgDuration());
        });
        
        if (overallSuccessRate > 0.95) {
            return Health.up().withDetails(details).build();
        } else {
            return Health.down().withDetails(details).build();
        }
    }
    
    /**
     * 队列指标
     */
    private static class QueueMetrics {
        private final AtomicLong successCount = new AtomicLong();
        private final AtomicLong failureCount = new AtomicLong();
        private final AtomicLong totalDuration = new AtomicLong();
        private final AtomicInteger consecutiveFailures = new AtomicInteger();
        
        synchronized void recordSuccess(long duration) {
            successCount.incrementAndGet();
            totalDuration.addAndGet(duration);
            consecutiveFailures.set(0);
        }
        
        synchronized void recordFailure(Exception e) {
            failureCount.incrementAndGet();
            consecutiveFailures.incrementAndGet();
        }
        
        QueueStats getStats() {
            QueueStats stats = new QueueStats();
            stats.setSuccessCount(successCount.get());
            stats.setFailureCount(failureCount.get());
            stats.setTotalCount(stats.getSuccessCount() + stats.getFailureCount());
            
            if (stats.getTotalCount() > 0) {
                stats.setSuccessRate((double) stats.getSuccessCount() / stats.getTotalCount());
            } else {
                stats.setSuccessRate(1.0);
            }
            
            if (stats.getSuccessCount() > 0) {
                stats.setAvgDuration(totalDuration.get() / stats.getSuccessCount());
            }
            
            stats.setConsecutiveFailures(consecutiveFailures.get());
            
            return stats;
        }
    }
    
    /**
     * 队列统计
     */
    @Data
    public static class QueueStats {
        private long successCount = 0;
        private long failureCount = 0;
        private long totalCount = 0;
        private double successRate = 1.0;
        private long avgDuration = 0;
        private int consecutiveFailures = 0;
    }
}
```

```
@Configuration
public class ProductionAckConfig {
    
    /**
     * 生产环境容器工厂

    @Bean
    public SimpleRabbitListenerContainerFactory productionContainerFactory(

     * 生产环境重试拦截器
     */
    private Advice productionRetryInterceptor() {
        return RetryInterceptorBuilder.stateless()
            .maxAttempts(3)
            .backOffOptions(1000, 2.0, 10000)
            .recoverer(new RejectAndDontRequeueRecoverer())
            .retryPolicy(retryPolicy())
            .build();
    }
    
    /**
     * 重试策略
     */
    private RetryPolicy retryPolicy() {
        Map<Class<? extends Throwable>, Boolean> retryableExceptions = new HashMap<>();
        retryableExceptions.put(IOException.class, true);
        retryableExceptions.put(TimeoutException.class, true);
        retryableExceptions.put(IllegalArgumentException.class, false); // 参数错误不重试
        
        return new SimpleRetryPolicy(3, retryableExceptions);
    }
    
    /**
     * 错误处理器
     */
    private ErrorHandler productionErrorHandler() {
        return new ConditionalRejectingErrorHandler(
            new ConditionalRejectingErrorHandler.DefaultExceptionStrategy() {
                @Override
                public boolean isFatal(Throwable t) {
                    return t instanceof MessageConversionException || 
                           t instanceof org.springframework.amqp.AmqpRejectAndDontRequeueException;
                }
            }
        );
    }
    
    /**
     * 生产环境消费者
     */
    @Service
    public class ProductionConsumer {
        
        @Autowired
        private AckMonitor ackMonitor;
        
        /**
         * 生产环境处理方法
         */
        @RabbitListener(queues = "production.queue", 
                       containerFactory = "productionContainerFactory")
        public void handleProductionMessage(String message,
                                           Channel channel,
                                           @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag,
                                           @Header(AmqpHeaders.CONSUMER_QUEUE) String queue) 
                throws IOException {
            
            String messageId = extractMessageId(message);
            
            // 监控处理
            ackMonitor.monitorAck(() -> {
                try {
                    // 业务处理
                    processBusiness(message);
                    
                    // 确认消息
                    channel.basicAck(deliveryTag, false);
                    
                    return null;
                    
                } catch (Exception e) {
                    // 处理失败
                    handleProcessingFailure(message, e, channel, deliveryTag, queue);
                    throw e;
                }
            }, queue, messageId);
        }
        
        /**
         * 处理失败逻辑
         */
        private void handleProcessingFailure(String message, Exception e, 
                                           Channel channel, long deliveryTag, 
                                           String queue) throws IOException {
            
            // 1. 记录日志
            logFailure(message, e, queue);
            
            // 2. 判断是否可重试
            if (isRecoverable(e)) {
                // 重新入队
                channel.basicNack(deliveryTag, false, true);
                System.err.println("消息重新入队: " + message);
            } else {
                // 发送到死信队列
                channel.basicNack(deliveryTag, false, false);
                System.err.println("发送到死信队列: " + message);
            }
            
            // 3. 发送告警
            if (shouldAlert(e)) {
                sendAlert(message, e, queue);
            }
        }
        
        /**
         * 批量处理
         */
        @RabbitListener(queues = "batch.production.queue", 
                       containerFactory = "productionContainerFactory")
        public void handleBatchProduction(List<Message> messages, Channel channel) 
                throws IOException {
            
            List<Long> successTags = new ArrayList<>();
            List<Long> failureTags = new ArrayList<>();
            
            for (Message message : messages) {
                long deliveryTag = message.getMessageProperties().getDeliveryTag();
                
                try {
                    String body = new String(message.getBody());
                    processBusiness(body);
                    successTags.add(deliveryTag);
                    
                } catch (Exception e) {
                    System.err.println("批量处理失败: " + e.getMessage());
                    failureTags.add(deliveryTag);
                }
            }
            
            // 批量确认成功
            if (!successTags.isEmpty()) {
                long lastTag = successTags.get(successTags.size() - 1);
                channel.basicAck(lastTag, true);
            }
            
            // 批量拒绝失败
            if (!failureTags.isEmpty()) {
                long lastTag = failureTags.get(failureTags.size() - 1);
                channel.basicNack(lastTag, true, true);
            }
        }
        
        private String extractMessageId(String message) {
            return Integer.toHexString(message.hashCode());
        }
        
        private void processBusiness(String message) throws Exception {
            // 业务处理逻辑
            if (message.contains("error")) {
                throw new Exception("模拟业务处理失败");
            }
            
            // 正常处理
            Thread.sleep(100);
        }
        
        private boolean isRecoverable(Exception e) {
            // 判断是否可恢复错误
            return !(e instanceof IllegalArgumentException);
        }
        
        private boolean shouldAlert(Exception e) {
            // 判断是否需要告警
            return e instanceof IOException || e instanceof TimeoutException;
        }
        
        private void logFailure(String message, Exception e, String queue) {
            // 记录失败日志
            System.err.println("处理失败 - 队列: " + queue + ", 消息: " + message);
        }
        
        private void sendAlert(String message, Exception e, String queue) {
            // 发送告警
            System.err.println("发送告警 - 队列: " + queue + ", 错误: " + e.getMessage());
        }
    }
```

## 七、 常见问题

### Q1: 手动确认 vs 自动确认？

- **手动确认**：消息处理成功后确认，保证可靠性
    
- **自动确认**：消息发送后立即确认，性能好但可能丢失消息
    
- **推荐**：生产环境使用手动确认
    

### Q2: 消息堆积怎么办？

- 增加消费者数量
    
- 提高预取数量
    
- 使用批量处理
    
- 监控队列长度
    

### Q3: 确认失败如何重试？

- 使用 basicNack 重新入队
    
- 配合死信队列
    
- 实现重试策略
    
- 监控重试次数
    

### Q4: 如何保证消息顺序？

- 单消费者+单队列
    
- 使用消息分组
    
- 注意重新入队可能破坏顺序
    

消费者确认机制是保证消息可靠性的关键，合理配置可以确保消息不丢失、不重复消费。
