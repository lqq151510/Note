# RabbitMQ 惰性队列（Lazy Queue）

惰性队列是 RabbitMQ 的高性能队列类型，将消息**直接存储在磁盘**而不是内存，避免消息在内存中堆积，显著降低内存使用，提高系统稳定性。

## 一、 核心特性

### 1. 工作模式对比

```
graph TD
    subgraph 标准队列
        A[消息到达] --> B[内存存储]
        B --> C[持久化到磁盘]
        C --> D[内存中维护索引]
    end
    
    subgraph 惰性队列
        E[消息到达] --> F[直接写入磁盘]
        F --> G[内存中仅保留元数据]
    end

    style B fill:#e1f5fe

    style D fill:#e1f5fe
    style E fill:#f1f8e9

    style G fill:#f1f8e9
```

### 2. 核心优势

- **内存保护**：消息直接存磁盘，避免内存爆炸
    
- **高吞吐**：批量磁盘写入，减少I/O次数
    
- **支持海量消息**：存储数百万甚至数千万消息
    
- **成本效益**：磁盘比内存更经济

```
public class LazyQueueConfig {

    @Bean
    public Queue lazyQueue() {
```

### 2. 惰性队列生产者

```
@Component
public class LazyQueueProducer {
    
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    /**
     * 发送消息到惰性队列
     */
    public void sendToLazyQueue(String message) {
        // 发送持久化消息
        rabbitTemplate.convertAndSend("lazy.exchange", "lazy.key", message, 
            message1 -> {
                MessageProperties props = message1.getMessageProperties();
                props.setDeliveryMode(MessageDeliveryMode.PERSISTENT);
                props.setHeader("queue-mode", "lazy");
                return message1;
            });
        
        System.out.println("发送到惰性队列: " + message);
    }
    
    /**
     * 批量发送测试
     */
    public void sendBatchToLazyQueue(int count) {
        for (int i = 1; i <= count; i++) {
            String message = "惰性队列消息#" + i + " - " + LocalDateTime.now();
            sendToLazyQueue(message);
            
            if (i % 1000 == 0) {
                System.out.println("已发送 " + i + " 条消息");
            }
            
            // 控制发送速率
            if (i % 10000 == 0) {
                try {
                    Thread.sleep(1000);
                } catch (InterruptedException e) {
                    Thread.currentThread().interrupt();
                }
            }
        }
    }
    
    /**
     * 发送大消息
     */
    public void sendLargeMessage() {
        // 生成1MB的大消息
        StringBuilder largeMessage = new StringBuilder();
        for (int i = 0; i < 1024 * 1024; i++) { // 1MB
            largeMessage.append("A");
        }
        
        sendToLazyQueue(largeMessage.toString());
        System.out.println("发送大消息完成，大小: 1MB");
    }
}
```

```
public class LazyQueueConsumer {

     * 基础消费者
     */
    @RabbitListener(queues = "lazy.queue")
    public void handleLazyQueue(String message,
                               @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag,
                               Channel channel) throws IOException {
        
        long startTime = System.currentTimeMillis();
        
        try {
            System.out.println("开始处理消息: " + message.substring(0, Math.min(50, message.length())));
            
            // 模拟处理时间
            Thread.sleep(100);
            
            // 确认消息
            channel.basicAck(deliveryTag, false);
            
            long duration = System.currentTimeMillis() - startTime;
            System.out.println("处理完成，耗时: " + duration + "ms");
            
        } catch (Exception e) {
            System.err.println("处理失败: " + e.getMessage());
            // 拒绝消息，不重新入队
            channel.basicNack(deliveryTag, false, false);
        }
    }
    
    /**
     * 批量消费者
     */
    @RabbitListener(queues = "lazy.queue", 
                   containerFactory = "batchContainerFactory")
    public void handleBatch(List<Message> messages, Channel channel) throws IOException {
        
        System.out.println("批量处理 " + messages.size() + " 条消息");
        
        for (Message message : messages) {
            try {
                String body = new String(message.getBody());
                System.out.println("处理消息: " + body.substring(0, Math.min(30, body.length())));
                
                // 批量确认
                long deliveryTag = message.getMessageProperties().getDeliveryTag();
                channel.basicAck(deliveryTag, false);
                
            } catch (Exception e) {
                System.err.println("批量处理失败: " + e.getMessage());
            }
        }
    }
    
    /**
     * 监控消费者
     */
    @RabbitListener(queues = "lazy.queue")
    public void handleWithMetrics(String message,
                                 @Header(AmqpHeaders.DELIVERY_TAG) long deliveryTag,
                                 @Header(AmqpHeaders.MESSAGE_COUNT) Integer remainingCount,
                                 Channel channel) throws IOException {
        
        if (remainingCount != null) {
            System.out.println("队列剩余消息: " + remainingCount);
        }
        
        // 处理逻辑
        channel.basicAck(deliveryTag, false);
    }
}
```

### 2. 惰性队列监控

```
@Component
public class LazyQueueMonitor {
    
    @Autowired
    private RabbitAdmin rabbitAdmin;
    
    /**
     * 获取队列状态
     */
    public QueueStats getQueueStats(String queueName) {
        return rabbitAdmin.getQueueInfo(queueName);
    }
    
    /**
     * 监控所有惰性队列
     */
    public Map<String, LazyQueueMetrics> monitorAllLazyQueues() {
        Map<String, LazyQueueMetrics> metrics = new HashMap<>();
        
        Properties queueProps = rabbitAdmin.getQueueProperties("lazy.queue");
        if (queueProps != null) {
            LazyQueueMetrics queueMetrics = extractQueueMetrics(queueProps);
            metrics.put("lazy.queue", queueMetrics);
        }
        
        return metrics;
    }
    
    /**
     * 自动清理监控
     */
    @Scheduled(fixedDelay = 300000) // 每5分钟执行
    public void autoCleanupMonitor() {
        monitorMemoryUsage();
        cleanupOldMessages();
        checkDiskSpace();
    }
    
    /**
     * 监控内存使用
     */
    private void monitorMemoryUsage() {
        Properties queueProps = rabbitAdmin.getQueueProperties("lazy.queue");
        if (queueProps != null) {
            Long messages = (Long) queueProps.get("messages");
            Long messagesReady = (Long) queueProps.get("messages_ready");
            
            System.out.println("队列监控:");
            System.out.println("总消息数: " + messages);
            System.out.println("就绪消息数: " + messagesReady);
            
            // 检查内存使用
            if (messagesReady > 10000) {
                System.out.println("警告: 就绪消息数较多，考虑增加消费者");
            }
        }
    }
    
    /**
     * 清理旧消息
     */
    private void cleanupOldMessages() {
        // 清理超过7天的消息
        Long ttl = 7 * 24 * 60 * 60 * 1000L; // 7天
        
        Map<String, Object> args = new HashMap<>();
        args.put("x-message-ttl", ttl);
        args.put("x-expires", ttl * 2);
        
        // 可以动态更新队列参数
        rabbitAdmin.declareQueue(new Queue("lazy.queue", true, false, false, args));
    }
    
    /**
     * 检查磁盘空间
     */
    private void checkDiskSpace() {
        File diskPartition = new File("/");
        long freeSpace = diskPartition.getFreeSpace();
        long totalSpace = diskPartition.getTotalSpace();
        double freePercent = (double) freeSpace / totalSpace * 100;
        
        if (freePercent < 20.0) {
            System.err.println("磁盘空间不足: " + String.format("%.1f", freePercent) + "% 空闲");
            // 发送告警
            sendDiskSpaceAlert(freePercent);
        }
    }
    
    private LazyQueueMetrics extractQueueMetrics(Properties props) {
        LazyQueueMetrics metrics = new LazyQueueMetrics();
        
        metrics.setQueueName((String) props.get("name"));
        metrics.setMessageCount((Long) props.get("messages"));
        metrics.setConsumerCount((Integer) props.get("consumers"));
        metrics.setMemoryBytes((Long) props.get("memory"));
        
        return metrics;
    }
    
    private void sendDiskSpaceAlert(double freePercent) {
        // 发送磁盘空间告警
        System.err.println("发送磁盘空间告警: " + freePercent + "% 空闲");
    }
    
    /**
     * 惰性队列指标
     */
    @Data
    public static class LazyQueueMetrics {
        private String queueName;
        private Long messageCount;
        private Integer consumerCount;
        private Long memoryBytes;
        private LocalDateTime checkTime = LocalDateTime.now();
    }
}
```

```
@Component
public class LogCollectorSystem {
    
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    /**
     * 配置日志收集队列
     */
    @Bean
    public Queue logQueue() {
        Map<String, Object> args = new HashMap<>();
        args.put("x-queue-mode", "lazy");
        args.put("x-max-length", 10000000);      // 最多1000万条日志
        args.put("x-message-ttl", 604800000);    // 7天过期
        args.put("x-overflow", "drop-head");     // 队列满时丢弃最旧日志
        
        return QueueBuilder.durable("log.collector.queue")
                .withArguments(args)
                .build();
    }
    
    /**
     * 收集日志
     */
    public void collectLog(String level, String service, String message) {
        LogEntry logEntry = new LogEntry();
        logEntry.setLevel(level);
        logEntry.setService(service);
        logEntry.setMessage(message);
        logEntry.setTimestamp(System.currentTimeMillis());
        logEntry.setHost(getHostName());
        
        // 发送到惰性队列
        rabbitTemplate.convertAndSend("log.exchange", "log.collect", logEntry,
            msg -> {
                msg.getMessageProperties().setDeliveryMode(MessageDeliveryMode.PERSISTENT);
                msg.getMessageProperties().setPriority(getLogPriority(level));
                return msg;
            });
    }
    
    /**
     * 批量收集日志
     */
    public void collectBatchLogs(List<LogEntry> logs) {
        for (LogEntry log : logs) {
            collectLog(log.getLevel(), log.getService(), log.getMessage());
        }
    }
    
    /**
     * 日志处理消费者
     */
    @RabbitListener(queues = "log.collector.queue", 
                   containerFactory = "batchContainerFactory")
    public void processLogs(List<Message> messages) {
        List<LogEntry> logs = new ArrayList<>();
        
        for (Message message : messages) {
            try {
                LogEntry log = deserializeLog(message.getBody());
                logs.add(log);
                
                // 批量写入到Elasticsearch
                if (logs.size() >= 100) {
                    saveToElasticsearch(logs);
                    logs.clear();
                }
            } catch (Exception e) {
                System.err.println("日志处理失败: " + e.getMessage());
            }
        }
        
        // 保存剩余日志
        if (!logs.isEmpty()) {
            saveToElasticsearch(logs);
        }
    }
    
    private int getLogPriority(String level) {
        switch (level.toUpperCase()) {
            case "ERROR": return 5;
            case "WARN": return 3;
            case "INFO": return 1;
            default: return 0;
        }
    }
    
    private String getHostName() {
        try {
            return InetAddress.getLocalHost().getHostName();
        } catch (UnknownHostException e) {
            return "unknown";
        }
    }
    
    private LogEntry deserializeLog(byte[] body) throws Exception {
        ObjectMapper mapper = new ObjectMapper();
        return mapper.readValue(body, LogEntry.class);
    }
    
    private void saveToElasticsearch(List<LogEntry> logs) {
        // 保存到Elasticsearch
        System.out.println("保存 " + logs.size() + " 条日志到ES");
    }
    
    /**
     * 日志条目
     */
    @Data
    public static class LogEntry {
        private String level;
        private String service;
        private String message;
        private long timestamp;
        private String host;
        private Map<String, String> tags = new HashMap<>();
    }
}
```

### 1. 读写优化策略

```
@Component
public class LazyQueuePerformanceOptimizer {
    
    /**
     * 写优化策略
     */
    public static class WriteOptimization {
        
        /**
         * 批量写入
         */
        public void batchWrite(RabbitTemplate template, List<?> messages, 
                              String exchange, String routingKey) {
            
            // 启用事务
            template.setChannelTransacted(true);
            
            template.execute(channel -> {
                try {
                    channel.txSelect();
                    
                    for (Object message : messages) {
                        template.convertAndSend(exchange, routingKey, message);
                    }
                    
                    channel.txCommit();
                    System.out.println("批量写入完成: " + messages.size() + " 条");
                    
                } catch (Exception e) {
                    channel.txRollback();
                    System.err.println("批量写入失败: " + e.getMessage());
                }
                return null;
            });
        }
        
        /**
         * 异步写入
         */
        public CompletableFuture<Void> asyncWrite(RabbitTemplate template, 
                                                 Object message, 
                                                 String exchange, 
                                                 String routingKey) {
            
            return CompletableFuture.runAsync(() -> {
                try {
                    template.convertAndSend(exchange, routingKey, message);
                } catch (Exception e) {
                    System.err.println("异步写入失败: " + e.getMessage());
                }
            });
        }
        
        /**
         * 压缩写入
         */
        public void writeWithCompression(RabbitTemplate template, 
                                        Object message, 
                                        String exchange, 
                                        String routingKey) {
            
            MessagePostProcessor compressor = msg -> {
                byte[] original = msg.getBody();
                byte[] compressed = compress(original);
                
                msg.getMessageProperties().setHeader("compressed", true);
                msg.getMessageProperties().setHeader("original-size", original.length);
                msg.getMessageProperties().setHeader("compressed-size", compressed.length);
                
                return new Message(compressed, msg.getMessageProperties());
            };
            
            template.convertAndSend(exchange, routingKey, message, compressor);
        }
        
        private byte[] compress(byte[] data) throws IOException {
            ByteArrayOutputStream baos = new ByteArrayOutputStream();
            try (GZIPOutputStream gzos = new GZIPOutputStream(baos)) {
                gzos.write(data);
            }
            return baos.toByteArray();
        }
    }
    
    /**
     * 读优化策略
     */
    public static class ReadOptimization {
        
        /**
         * 批量读取
         */
        public List<Message> batchRead(Channel channel, String queueName, 
                                      int batchSize, long timeoutMillis) 
                throws IOException, InterruptedException {
            
            List<Message> messages = new ArrayList<>();
            GetResponse response;
            
            long endTime = System.currentTimeMillis() + timeoutMillis;
            
            do {
                response = channel.basicGet(queueName, false);
                if (response != null) {
                    messages.add(new Message(
                        response.getBody(),
                        new MessageProperties()
                    ));
                    
                    // 确认消息
                    channel.basicAck(response.getEnvelope().getDeliveryTag(), false);
                }
                
                if (messages.size() >= batchSize) {
                    break;
                }
                
                if (System.currentTimeMillis() > endTime) {
                    break;
                }
                
                Thread.sleep(10);
                
            } while (response != null);
            
            return messages;
        }
        
        /**
         * 并行消费
         */
        public void parallelConsume(String queueName, int parallelLevel, 
                                   Consumer<Message> processor) {
            
            ExecutorService executor = Executors.newFixedThreadPool(parallelLevel);
            List<CompletableFuture<Void>> futures = new ArrayList<>();
            
            for (int i = 0; i < parallelLevel; i++) {
                CompletableFuture<Void> future = CompletableFuture.runAsync(() -> {
                    try {
                        consumeInParallel(queueName, processor);
                    } catch (Exception e) {
                        System.err.println("并行消费失败: " + e.getMessage());
                    }
                }, executor);
                futures.add(future);
            }
            
            // 等待所有完成
            CompletableFuture.allOf(futures.toArray(new CompletableFuture[0]))
                .join();
            
            executor.shutdown();
        }
        
        private void consumeInParallel(String queueName, Consumer<Message> processor) 
                throws Exception {
            
            ConnectionFactory factory = new ConnectionFactory();
            factory.setHost("localhost");
            
            try (Connection connection = factory.newConnection();
                 Channel channel = connection.createChannel()) {
                
                channel.basicQos(100); // 预取数量
                
                GetResponse response;
                while ((response = channel.basicGet(queueName, false)) != null) {
                    Message message = new Message(
                        response.getBody(),
                        new MessageProperties()
                    );
                    
                    try {
                        processor.accept(message);
                        channel.basicAck(response.getEnvelope().getDeliveryTag(), false);
                    } catch (Exception e) {
                        channel.basicNack(response.getEnvelope().getDeliveryTag(), false, true);
                    }
                }
            }
        }
    }
    
    /**
     * 内存优化策略
     */
    public static class MemoryOptimization {
        
        /**
         * 监控内存使用
         */
        public void monitorMemoryUsage(String queueName) {
            ScheduledExecutorService scheduler = Executors.newSingleThreadScheduledExecutor();
            
            scheduler.scheduleAtFixedRate(() -> {
                try {
                    Runtime runtime = Runtime.getRuntime();
                    long usedMemory = runtime.totalMemory() - runtime.freeMemory();
                    long maxMemory = runtime.maxMemory();
                    double usagePercent = (double) usedMemory / maxMemory * 100;
                    
                    System.out.println("内存使用: " + 
                        String.format("%.1f", usagePercent) + "% (" + 
                        formatBytes(usedMemory) + " / " + 
                        formatBytes(maxMemory) + ")");
                    
                    if (usagePercent > 80.0) {
                        System.err.println("内存使用过高，考虑增加惰性队列或减少内存队列");
                        triggerMemoryCleanup();
                    }
                    
                } catch (Exception e) {
                    System.err.println("内存监控失败: " + e.getMessage());
                }
            }, 0, 30, TimeUnit.SECONDS);
        }
        
        /**
         * 触发内存清理
         */
        private void triggerMemoryCleanup() {
            System.gc(); // 触发垃圾回收
            System.out.println("触发内存清理");
        }
        
        private String formatBytes(long bytes) {
            if (bytes < 1024) return bytes + " B";
            int exp = (int) (Math.log(bytes) / Math.log(1024));
            String pre = "KMGTPE".charAt(exp-1) + "";
            return String.format("%.1f %sB", bytes / Math.pow(1024, exp), pre);
        }
    }
```

```
@Component

    /**
     * 监控单个队列
     */
    private void monitorQueue(String queueName) {
        Properties props = rabbitAdmin.getQueueProperties(queueName);
        if (props != null) {
            QueueMetrics metrics = extractMetrics(queueName, props);
            queueMetrics.put(queueName, metrics);
            
            // 更新指标
            updateMeterRegistry(queueName, metrics);
            
            // 记录到日志
            logMetrics(queueName, metrics);
        }
    }
    
    /**
     * 检查阈值
     */
    private void checkThresholds() {
        queueMetrics.forEach((queueName, metrics) -> {
            // 检查消息积压
            if (metrics.getMessageCount() > 100000) {
                sendAlert(queueName, "消息积压", 
                    "队列 " + queueName + " 消息数: " + metrics.getMessageCount());
            }
            
            // 检查内存使用
            if (metrics.getMemory() > 100 * 1024 * 1024) { // 100MB
                sendAlert(queueName, "内存使用高", 
                    "队列 " + queueName + " 内存使用: " + formatBytes(metrics.getMemory()));
            }
            
            // 检查消费者数量
            if (metrics.getConsumerCount() == 0 && metrics.getMessageCount() > 0) {
                sendAlert(queueName, "无消费者", 
                    "队列 " + queueName + " 有消息但无消费者");
            }
        });
    }
    
    /**
     * 报告指标
     */
    private void reportMetrics() {
        Map<String, Object> report = new HashMap<>();
        
        queueMetrics.forEach((queueName, metrics) -> {
            Map<String, Object> queueReport = new HashMap<>();
            queueReport.put("messageCount", metrics.getMessageCount());
            queueReport.put("consumerCount", metrics.getConsumerCount());
            queueReport.put("memoryBytes", metrics.getMemory());
            queueReport.put("rate", metrics.getRate());
            
            report.put(queueName, queueReport);
        });
        
        // 发送到监控系统
        sendToMonitoringSystem(report);
    }

                () -> queueMetrics.getOrDefault("lazy.queue", new QueueMetrics()).getMessageCount())
            .description("惰性队列消息数")
            .register(meterRegistry);
        
        Gauge.builder("rabbitmq.lazy.queue.memory",

            .description("惰性队列内存使用")
            .register(meterRegistry);
    }
```

## 八、 常见问题

### Q1: 惰性队列 vs 标准队列？

|特性|惰性队列|标准队列|
|---|---|---|
|存储位置|直接磁盘|内存 + 磁盘|
|内存使用|低|高|
|适合场景|海量消息、大数据|实时消息、低延迟|
|性能|高吞吐|低延迟|
|成本|低|高|

### Q2: 何时使用惰性队列？

- 日志收集系统
    
- 大数据处理管道
    
- 消息归档系统
    
- 高吞吐量场景
    
- 内存受限环境
    

### Q3: 性能优化建议？

- 使用批量操作
    
- 适当增加消费者
    
- 启用压缩
    
- 监控磁盘I/O
    
- 定期清理旧数据
    

### Q4: 监控重点？

- 磁盘使用率
    
- 队列长度增长
    
- 消费者处理速度
    
- 内存使用情况
    
- 消息过期情况
    

惰性队列是处理海量数据的利器，合理使用可以显著降低内存压力，提高系统稳定性。
