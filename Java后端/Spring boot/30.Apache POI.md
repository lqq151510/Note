Apache POI（Poor Obfuscation Implementation）是 Apache 软件基金会维护的开源 Java 库，用于读写和操作 Microsoft Office 格式文件，包括 Excel、Word、PowerPoint 等，无需安装 Office 软件。

## 核心功能与特点

- **跨平台支持**：基于 Java 开发，可在 Windows、Linux、macOS 等系统运行
    
- **格式兼容**：支持旧版二进制格式（.xls、.doc、.ppt）和现代 OOXML 格式（.xlsx、.docx、.pptx）
    
- **丰富操作**：可创建、读取、修改单元格/段落/幻灯片内容，设置样式、公式、图表、超链接等
    
- **高性能处理**：提供 SXSSF 模块支持流式写入，优化百万级数据导出时的内存使用
    

## 主要模块架构

|模块|支持格式|功能说明|
|---|---|---|
|**HSSF**​|Excel 97-2003 (.xls)|基础读写、公式计算|
|**XSSF**​|Excel 2007+ (.xlsx)|XML 格式解析、大数据支持|
|**SXSSF**​|Excel 2007+ (.xlsx)|流式写入、内存优化|
|**HWPF**​|Word 97-2003 (.doc)|段落/表格操作、样式继承|
|**XWPF**​|Word 2007+ (.docx)|文档结构解析、内容提取|
|**HSLF**​|PowerPoint 97-2003 (.ppt)|幻灯片元素操作|
|**XSLF**​|PowerPoint 2007+ (.pptx)|动画/过渡效果控制|

## 典型应用场景

1. **报表自动化生成**：从数据库读取数据自动生成 Excel/Word 报表
    
2. **数据导入导出**：Web 应用中处理用户上传/下载的 Office 文件
    
3. **文档转换服务**：将 Office 文档转换为 PDF 等其他格式
    
4. **批量数据处理**：读取 Excel 文件进行数据分析，结果写回文档

## 快速开始示例

```
// 读取 Excel 文件示例
import org.apache.poi.xssf.usermodel.XSSFWorkbook;
import org.apache.poi.ss.usermodel.*;
import java.io.FileInputStream;

public class ReadExcel {
    public static void main(String[] args) throws Exception {
        Workbook workbook = new XSSFWorkbook(new FileInputStream("data.xlsx"));
        Sheet sheet = workbook.getSheetAt(0);
        Row row = sheet.getRow(0);
        Cell cell = row.getCell(0);
        System.out.println(cell.getStringCellValue());
        workbook.close();
    }
}
```

官方文档和最新版本可在 [poi.apache.org](https://poi.apache.org/)获取。对于新项目开发，推荐使用 5.5.1 版本以获得最佳安全性和功能支持。
