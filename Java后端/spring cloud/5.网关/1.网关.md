网关是所有网络流量的统一入口和调度中心，就像公司的前台总机。

## 核心定义

网关是位于客户端和后端服务之间的中间层，负责处理所有入站请求的路由、过滤和转换。它是微服务架构中的“门卫”，统一管理外部访问。

## 关键特性

**统一接入点**：所有外部请求都通过网关进入系统，客户端只需知道网关地址。

**请求路由**：根据路径、域名等规则，将请求转发到对应的后端服务。

**安全防护**：统一处理身份认证、权限校验、防攻击等安全措施。

**流量管控**：实现限流、熔断、负载均衡，保护后端服务。

**协议转换**：支持HTTP、WebSocket、gRPC等多种协议转换。

**监控日志**：集中收集访问日志、性能指标，便于监控分析。

## 常见产品

- **Spring Cloud Gateway**：基于Spring生态的响应式API网关
    
- **Kong**：基于Nginx的高性能开源网关
    
- **Nginx**：最常用的反向代理和负载均衡器
    
- **API Gateway**：云服务商提供的托管网关服务
    

简单说，网关让客户端访问更简单，让后端服务更安全，是微服务架构的关键基础设施。
