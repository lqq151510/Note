### 2. 创建文档 (Create/Index)

#### 方式一：指定文档ID

```
public void createDocument(RestHighLevelClient client) throws IOException {
    // 1. 准备数据
    IndexRequest request = new IndexRequest("users");
    request.id("1001");  // 指定文档ID
    
    // 构造文档内容（JSON格式）
    String json = "{" +
            "\"name\":\"张三\"," +
            "\"age\":25," +
            "\"gender\":\"男\"," +
            "\"hobby\":[\"篮球\", \"游泳\", \"编程\"]," +
            "\"address\":{" +
            "\"city\":\"北京\"," +
            "\"street\":\"中关村大街\"" +
            "}" +
            "}";
    
    request.source(json, XContentType.JSON);
    
    // 2. 执行请求
    IndexResponse response = client.index(request, RequestOptions.DEFAULT);
    
    // 3. 处理响应
    System.out.println("文档ID: " + response.getId());
    System.out.println("索引名称: " + response.getIndex());
    System.out.println("操作结果: " + response.getResult()); // CREATED
}
```

#### 方式二：不指定文档ID（自动生成）

```
public void createDocumentAutoId(RestHighLevelClient client) throws IOException {

    // 不指定ID，ES会自动生成
    
    Map<String, Object> jsonMap = new HashMap<>();

    jsonMap.put("age", 30);
    jsonMap.put("create_time", "2024-01-15");
    
    request.source(jsonMap);
    
    IndexResponse response = client.index(request, RequestOptions.DEFAULT);
    System.out.println("自动生成的文档ID: " + response.getId());
}
```

### 3. 查询文档 (Read)

#### 3.1 根据ID查询单个文档

```
public void getDocumentById(RestHighLevelClient client) throws IOException {
    GetRequest request = new GetRequest("users", "1001");
    
    GetResponse response = client.get(request, RequestOptions.DEFAULT);
    
    if (response.isExists()) {
        // 获取文档内容
        String sourceAsString = response.getSourceAsString();
        System.out.println("文档内容: " + sourceAsString);
        
        // 获取特定字段
        Map<String, Object> source = response.getSource();
        System.out.println("姓名: " + source.get("name"));
        System.out.println("年龄: " + source.get("age"));
    } else {
        System.out.println("文档不存在");
    }
}
```

```
        System.out.println("文档内容: " + hit.getSourceAsString());
        System.out.println("得分: " + hit.getScore());
    }
```

### 6. 批量操作 (Bulk)

```
public void bulkOperations(RestHighLevelClient client) throws IOException {
    BulkRequest request = new BulkRequest();
    
    // 1. 添加文档1
    Map<String, Object> doc1 = new HashMap<>();
    doc1.put("name", "王五");
    doc1.put("age", 35);
    request.add(new IndexRequest("users").id("2001").source(doc1));
    
    // 2. 更新文档
    Map<String, Object> updateFields = new HashMap<>();
    updateFields.put("age", 29);
    request.add(new UpdateRequest("users", "2001").doc(updateFields));
    
    // 3. 删除文档
    request.add(new DeleteRequest("users", "3001"));
    
    // 4. 执行批量操作
    BulkResponse responses = client.bulk(request, RequestOptions.DEFAULT);
    
    // 处理结果
    if (responses.hasFailures()) {
        System.out.println("批量操作有失败: " + responses.buildFailureMessage());
    }
    
    Arrays.stream(responses.getItems()).forEach(item -> {
        System.out.println("操作类型: " + item.getOpType());
        System.out.println("文档ID: " + item.getId());
        System.out.println("状态: " + item.status());
    });
}
```

### 7. 完整工具类示例

```
public class ElasticsearchCRUD {
    private RestHighLevelClient client;
    
    public ElasticsearchCRUD(RestHighLevelClient client) {
        this.client = client;
    }
    
    // 创建文档
    public String createDocument(String index, String id, Map<String, Object> source) 
            throws IOException {
        IndexRequest request = new IndexRequest(index);
        if (id != null) {
            request.id(id);
        }
        request.source(source);
        
        IndexResponse response = client.index(request, RequestOptions.DEFAULT);
        return response.getId();
    }
    
    // 根据ID查询
    public Map<String, Object> getDocument(String index, String id) 
            throws IOException {
        GetRequest request = new GetRequest(index, id);
        GetResponse response = client.get(request, RequestOptions.DEFAULT);
        
        if (response.isExists()) {
            return response.getSource();
        }
        return null;
    }
    
    // 更新文档
    public void updateDocument(String index, String id, Map<String, Object> updates) 
            throws IOException {
        UpdateRequest request = new UpdateRequest(index, id);
        request.doc(updates);
        request.docAsUpsert(true);  // 如果不存在则创建
        
        client.update(request, RequestOptions.DEFAULT);
    }
    
    // 删除文档
    public boolean deleteDocument(String index, String id) 
            throws IOException {
        DeleteRequest request = new DeleteRequest(index, id);
        DeleteResponse response = client.delete(request, RequestOptions.DEFAULT);
        
        return response.getResult() == DocWriteResponse.Result.DELETED;
    }
    
    // 批量插入
    public void bulkInsert(String index, List<Map<String, Object>> documents) 
            throws IOException {
        BulkRequest request = new BulkRequest();
        
        for (Map<String, Object> doc : documents) {
            request.add(new IndexRequest(index).source(doc));
        }
        
        BulkResponse response = client.bulk(request, RequestOptions.DEFAULT);
        
        if (response.hasFailures()) {
            throw new RuntimeException("批量插入失败: " + response.buildFailureMessage());
        }
    }
    
    // 关闭客户端
    public void close() throws IOException {
        if (client != null) {
            client.close();
        }
    }
}
```

### 8. 注意事项

1. **版本兼容性**：确保客户端版本与ES服务端版本一致
    
2. **连接管理**：使用连接池，避免频繁创建/销毁连接
    
3. **异常处理**：所有IO操作都应处理异常
    
4. **资源释放**：使用finally块或try-with-resources确保连接关闭
    
5. **批量操作**：大数据量操作使用bulk API，提高性能
    
6. **异步操作**：高并发场景考虑使用异步客户端
