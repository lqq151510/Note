IK分词器（IK Analyzer）是专门针对中文文本设计的**开源分词器**，也是目前**Elasticsearch**和**Lucene**生态中最主流的中文分词解决方案。

### 1. 核心功能：它解决了什么问题？

中文不像英文有天然的空格分隔，计算机无法直接识别词语。IK分词器通过内置的词典和算法，将连续的中文字符串切分成一个个有意义的词语。

- **智能分词**：结合词典和上下文语义，识别新词、专有名词，避免机械切分。
    
- **细粒度切分**：支持“细粒度”和“最细粒度”两种模式，适应不同搜索需求。
    

### 2. 两种核心模式

IK分词器提供了两种不同的切分策略，适用于不同的业务场景：

|模式|原理|示例（输入：中华人民共和国）|适用场景|
|---|---|---|---|
|**ik_smart**​|**智能切分**  <br>只输出最长的、最粗粒度的词语。|`中华人民共和国`|**搜索场景**  <br>减少索引体积，提高召回率。|
|**ik_max_word**​|**最细切分**  <br>穷尽所有可能的词语组合。|`中华人民共和国`  <br>`中华人民`  <br>`中华`  <br>`华人`  <br>`人民`  <br>`共和国`  <br>`共和`  <br>`国`|**索引场景**  <br>覆盖所有可能的搜索关键词，提高查全率。|

### 3. 核心优势

- **开源免费**：基于Apache 2.0协议，社区活跃，文档丰富。
    
- **高性能**：采用字典树（Trie Tree）结构，查询效率高。
    
- **可扩展**：支持自定义扩展词典和停用词典，适应特定行业术语。
    

### 4. 典型应用场景

- **搜索引擎**：作为Elasticsearch的默认中文分词插件，处理中文搜索。
    
- **日志分析**：在ELK（Elasticsearch, Logstash, Kibana）技术栈中分析中文日志。
    
- **内容检索**：网站、APP的内容搜索功能。
