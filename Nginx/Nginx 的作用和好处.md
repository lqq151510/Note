## Nginx 的作用和好处

**Nginx 的核心作用**：

1. **Web 服务器**：处理 HTTP/HTTPS 请求，提供静态内容服务
    
2. **反向代理**：接收客户端请求，转发到后端服务器，隐藏真实服务器信息
    
3. **负载均衡**：将流量分发到多个后端服务器，提高系统可用性和性能
    
4. **缓存加速**：缓存静态内容和动态内容，减少后端压力
    
5. **SSL/TLS 终端**：处理 HTTPS 加密解密，减轻后端服务器负担
    

**主要好处**：

- **高性能**：采用事件驱动架构，能处理数万并发连接，内存占用低
    
- **高可靠性**：7×24 小时稳定运行，支持热部署和配置重载
    
- **扩展性强**：模块化设计，支持 Lua、JavaScript 等脚本扩展
    
- **配置灵活**：简洁的配置文件语法，支持条件判断和变量
    
- **成本效益**：开源免费，替代商业负载均衡器如 F5、A10
    

**典型应用场景**：

- 网站动静分离（静态文件由 Nginx 直接处理）
    
- API 网关和微服务入口
    
- CDN 边缘节点
    
- 流媒体服务器（支持 RTMP、HLS）
    
- 安全防护（配合 WAF 模块）
    

Nginx 因其轻量、高效的特点，已成为现代 Web 架构中的核心组件，尤其适合高并发、低延迟的应用场景。
