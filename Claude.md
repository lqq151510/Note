# Claude 配置文档

## 项目说明

这是一个 Obsidian 笔记库，用于整理和存储各类技术文档、学习笔记和参考资料。

## 目录结构

- `Claude Code` - Claude Code 相关的笔记
- `Docker` - Docker 容器化部署笔记
- `Excalidraw` - Excalidraw 绘图笔记
- `Java后端` - Java 后端开发笔记
- `Linux` - Linux 系统相关笔记
- `Nginx` - Nginx 反向代理和配置笔记
- `leecode` - LeetCode 算法题解
- `前端` - 前端开发笔记
- `大模型` - 大语言模型（LLM）相关笔记
- `数据库` - 数据库相关笔记

## 使用说明

### 对于 AI 助手 (Claude)

1. **文件浏览**：
   - 这是一个静态的笔记库，所有内容都存储在本地
   - 可以浏览和搜索 `.md` 文件
   - 可以读取和理解笔记内容

2. **编辑建议**：
   - 遇到笔记内容不准确时，可以建议更新
   - 可以帮助整理笔记结构
   - 可以根据笔记内容生成相关的代码示例或文档

3. **限制**：
   - 无法访问真实的 Obsidian 同步服务
   - 无法执行 Obsidian 的自动化脚本
   - 所有编辑建议需要在确认后手动应用

### 对于用户

1. **如何使用**：
   - 这是一个普通的文件夹，直接用文本编辑器或 Markdown 编辑器打开
   - 推荐使用 Obsidian 客户端以获得最佳体验
   - 可以创建、编辑、搜索笔记

2. **组织方式**：
   - 按主题分类存储
   - 每个目录对应一个技术领域
   - Markdown 格式记录

3. **协作方式**：
   - 支持标准的 Git 版本控制
   - 支持多人协作编辑（如果使用 Obsidian Sync 或 Git）

## 技术栈

- **格式**：Markdown (.md)
- **版本控制**：Git
- **绘图**：Excalidraw（支持导出 SVG）
- **插件支持**：Obsidian 插件生态系统

## 备注

这是一个个人的知识库，记录了在学习和开发过程中的各种笔记、代码片段和参考资料。欢迎随时更新和补充内容。
