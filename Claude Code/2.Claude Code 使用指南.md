## 简介

**Claude Code** 是 Anthropic 官方推出的命令行 AI 助手工具，可以直接在终端中与代码库交互，帮助编写、调试和重构代码。

### 主要特性
- 🎯 直接操作代码文件
- 🔍 理解整个代码库结构
- 💻 执行终端命令
- 🔧 自动修复错误
- 📝 生成测试用例
- 🚀 代码重构和优化

## 安装

### 前置要求

1. **Node.js** (版本 18 或更高)
2. **npm** 包管理器

### 检查环境

```bash
node -v
npm -v
```

### 安装步骤

#### Windows 系统

```bash
# 1. 安装 Node.js (如未安装)
# 访问 https://nodejs.org/ 下载 LTS 版本

# 2. 全局安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 3. 验证安装
claude --version
```

#### macOS / Linux

```bash
# 使用 Homebrew (macOS)
brew install claude-code

# 或使用 npm
npm install -g @anthropic-ai/claude-code

# 验证安装
claude --version
```

## 配置

### 首次使用认证

运行以下命令启动 Claude Code：

```bash
claude
```

首次运行会提示你进行认证，按照指引完成登录。

### 配置文件位置

**Windows:**
```
C:\Users\<你的用户名>\.claude\settings.json
```

**macOS / Linux:**
```
~/.claude/settings.json
```

### 配置第三方模型（可选）

如果你想使用第三方模型（如 DeepSeek、Kimi 等），可以修改配置文件：

```json
{
  "apiProvider": "openai",
  "apiBaseUrl": "https://api.deepseek.com",
  "apiKey": "your-api-key",
  "model": "deepseek-chat"
}
```

### 环境变量配置

```bash
# Windows (PowerShell)
$env:ANTHROPIC_API_KEY="your-api-key"

# macOS / Linux
export ANTHROPIC_API_KEY="your-api-key"
```

## 基本使用

### 启动 Claude Code

```bash
# 在当前目录启动
claude

# 在指定目录启动
claude /path/to/project

# 带提示词启动
claude "帮我重构这个函数的代码"
```

### 基本交互

启动后，你可以：

1. **询问代码问题**
   ```
   这个函数的作用是什么？
   ```

2. **请求修改代码**
   ```
   把这个函数改成异步的
   ```

3. **让 Claude 执行命令**
   ```
   运行测试用例
   ```

4. **创建新文件**
   ```
   创建一个用户登录的组件
   ```

## 常用命令

### 内置命令

在 Claude Code 对话中，可以使用以下命令：

| 命令 | 说明 |
|------|------|
| `/clear` | 清空当前对话历史 |
| `/help` | 显示帮助信息 |
| `/exit` | 退出 Claude Code |
| `/tokens` | 显示当前使用的 token 数量 |
| `/model` | 查看或切换模型 |

### 快捷操作

- **Ctrl+C** - 中断当前操作
- **Ctrl+D** - 退出程序
- **Tab** - 自动补全

## 最佳实践

### ✅ 推荐做法

1. **明确具体的请求**
   - ❌ 差： "修复这个"
   - ✅ 好： "修复第 35 行的空指针异常"

2. **提供上下文**
   ```
   我正在开发一个电商网站，需要创建一个购物车功能...
   ```

3. **分步骤进行复杂任务**
   ```
   第一步：先创建数据库模型
   第二步：创建 API 路由
   第三步：编写测试用例
   ```

4. **审查生成的代码**
   - 始终检查 Claude 生成的代码
   - 运行测试确保功能正常

5. **使用版本控制**
   ```bash
   # 在让 Claude 修改前提交当前状态
   git add .
   git commit -m "before claude refactor"
   ```

### ⚠️ 注意事项

1. **安全性**
   - 不要提交 API Key 到代码库
   - 审查所有文件修改
   - 注意敏感信息泄露

2. **性能考虑**
   - 大文件可能消耗大量 token
   - 复杂任务分多次进行

3. **准确性**
   - Claude 可能犯错，需要人工审核
   - 重要代码需要测试验证

## 常见问题

### Q1: 安装失败怎么办？

```bash
# 清理 npm 缓存
npm cache clean --force

# 重新安装
npm install -g @anthropic-ai/claude-code
```

### Q2: 认证失败？

- 检查网络连接
- 确认 API Key 有效
- 查看官方文档获取最新认证方式

### Q3: 如何节省 token 使用？

- 使用简洁明确的提示词
- 避免重复相同问题
- 定期使用 `/clear` 清空历史

### Q4: 可以离线使用吗？

❌ Claude Code 需要联网使用，因为需要调用云端 API

### Q5: 支持哪些编程语言？

✅ 支持所有主流编程语言：
- JavaScript/TypeScript
- Python
- Java
- C/C++
- Go
- Rust
- 等等...

### Q6: 如何查看使用费用？

登录 Anthropic 官方控制台查看 API 使用情况和费用明细。
