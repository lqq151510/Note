**题目：只出现一次的数字**​

**描述**​

给定一个非空整数数组 `nums`，除某个元素仅出现一次外，其余每个元素均恰好出现两次。请找出那个只出现一次的元素。

**要求**​

- 设计并实现一个时间复杂度为 O(n) 的算法。
    
- 算法只能使用常量额外空间（即 O(1) 空间复杂度）。
    

**解题思路**​

利用异或运算的性质：

- 任何数与 0 异或结果为其自身：`a ^ 0 = a`
    
- 任何数与自身异或结果为 0：`a ^ a = 0`
    
- 异或运算满足交换律和结合律
    

因此，将数组中所有数字进行异或操作，出现两次的数字异或结果为 0，最终结果即为只出现一次的数字。

**代码实现（Java）**​

```
class Solution {
    public int singleNumber(int[] nums) {
        int result = 0;
        for (int num : nums) {
            result ^= num;
        }
        return result;
    }
}
```

（注：用户提供的流式写法同样正确，但循环版本更直观且符合常量空间要求。）

**复杂度分析**​

- 时间复杂度：O(n)，只需遍历数组一次。
    
- 空间复杂度：O(1)，仅使用常数变量。
