```
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        // 初始化两个指针，prev指向已反转部分的新头节点，curr指向待反转部分的头节点
        ListNode prev = null;
        ListNode curr = head;
        
        // 遍历原链表
        while (curr != null) {
            // 1. 先保存当前节点的下一个节点，防止断链
            ListNode nextTemp = curr.next;
            // 2. 核心操作：将当前节点的next指针指向前一个节点prev，完成反转
            curr.next = prev;
            // 3. 移动指针，为下一个节点的反转做准备
            prev = curr; // prev前移
            curr = nextTemp; // curr前移
        }
        // 当循环结束时，curr为null，prev正好指向原链表的最后一个节点，即新链表的头节点
        return prev;
    }
}
```

- **核心逻辑**：在遍历链表时，逐个改变节点的指向。关键在于，在修改`curr.next`前，必须先用临时变量`nextTemp`保存下一个节点，否则会丢失对后续链表的访问。
    
- **时间复杂度**：O(n)，需要遍历整个链表一次。
    
- **空间复杂度**：O(1)，只使用了几个固定的指针变量，是空间最优的解法。

### 💡 其他解法思路

除了您使用的迭代法，另一种常见的解法是**递归法**，它体现了分治的思想。

```
public ListNode reverseList(ListNode head) {
    // 递归终止条件：当前节点为空或已经是最后一个节点
    if (head == null || head.next == null) {
        return head;
    }
    // 递归反转以head.next开头的子链表
    ListNode newHead = reverseList(head.next);
    // 关键步骤：将当前节点接在已反转子链表的末尾
    head.next.next = head;
    head.next = null; // 断开原顺序的指针，防止成环
    return newHead; // 新的头节点始终是原链表的尾节点
}
```

- **思路**：假设头节点`head`之后的子链表已经被成功反转（`newHead`是子链表反转后的头），那么只需要将`head`节点接在已反转子链表的末尾，并断开`head`原来的连接即可。
    
- **复杂度**：时间复杂度同样为O(n)。空间复杂度为O(n)，主要用于递归调用栈。
