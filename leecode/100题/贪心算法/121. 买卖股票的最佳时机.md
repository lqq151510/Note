### 题目描述

给定一个数组 `prices`，其中 `prices[i]`表示一支给定股票第 `i`天的价格。你只能选择 **某一天**​ 买入这只股票，并选择在 **未来的某一个不同的日子**​ 卖出该股票。设计一个算法来计算你所能获取的最大利润。如果你不能获取任何利润，返回 `0`。

---

### 解题思路

1. 遍历数组，同时记录**当前遇到的最小股价**。
    
2. 对于每一天的价格：
    
    - 如果当前价格低于已知最低价，更新最低价。
        
    - 否则，计算以当前价格卖出可获得的利润（当前价格 - 最低价），并更新最大利润。
        
    
3. 遍历结束后，返回最大利润。
    

---

### 代码实现

```
class Solution {
    public int maxProfit(int[] prices) {
        if (prices == null || prices.length <= 1) {
            return 0;
        }
        int minPrice = prices[0];  // 记录当前最小股价
        int maxProfit = 0;         // 记录最大利润
        for (int i = 1; i < prices.length; i++) {
            int currentPrice = prices[i];
            if (currentPrice < minPrice) {
                minPrice = currentPrice;  // 更新最低价
            } else {
                int profit = currentPrice - minPrice;  // 计算当前利润
                maxProfit = Math.max(profit, maxProfit);  // 更新最大利润
            }
        }
        return maxProfit;
    }
}
```

---

### 复杂度分析

- **时间复杂度**：O(n)，只需遍历一次数组。
    
- **空间复杂度**：O(1)，只使用了常数个变量。
    

---

### 示例

```
输入: prices = [7,1,5,3,6,4]
输出: 5
解释: 在第 2 天买入（价格 = 1），在第 5 天卖出（价格 = 6），利润 = 6-1 = 5。
```
