题目：实现一个支持常数时间获取最小值的栈

设计一个支持 push，pop，top 操作，并能在常数时间内检索到最小元素的栈。

实现 `MinStack`类:

- `MinStack()`初始化堆栈对象。
    
- `void push(int val)`将元素 `val`推入堆栈。
    
- `void pop()`删除堆栈顶部的元素。
    
- `int top()`获取堆栈顶部的元素。
    
- `int getMin()`获取堆栈中的最小元素。
    

代码实现：

```
import java.util.Deque;
import java.util.ArrayDeque; 

class MinStack {
    private Deque<Integer> maindeque; // 主栈，用于存储所有元素
    private Deque<Integer> mindeque; // 辅助栈，栈顶始终存储当前主栈中的最小值

    public MinStack() {
        maindeque = new ArrayDeque<>();
        mindeque = new ArrayDeque<>();
    }
    
    public void push(int val) {
        maindeque.push(val);
        // 如果辅助栈为空，直接压入当前值；否则压入当前值与辅助栈栈顶值（当前最小值）的较小值
        if(mindeque.isEmpty()) {
            mindeque.push(val);
        } else {
            mindeque.push(Math.min(val, mindeque.peek()));
        }
    }
    
    public void pop() {
        maindeque.pop();
        mindeque.pop(); // 同步弹出，以保持两个栈状态一致
    }
    
    public int top() {
        return maindeque.peek();
    }
    
    public int getMin() {
        return mindeque.peek(); // 常数时间获取当前最小值
    }
}
```
