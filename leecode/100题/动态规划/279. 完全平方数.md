
### 问题描述

给定一个整数 `n`，返回和为 `n`的**完全平方数**的最少数量。

**完全平方数**是一个整数，其值等于另一个整数的平方。换句话说，其值等于一个整数自乘的积。例如，1、4、9 和 16 都是完全平方数，而 3 和 11 不是。

**示例 1：**

```
输入：n = 12
输出：3
解释：12 = 4 + 4 + 4
```

**示例 2：**

```
输入：n = 13
输出：2
解释：13 = 4 + 9
```

### 解题思路

这是一个典型的动态规划问题，类似于**完全背包问题**，其中：

- 物品是所有的完全平方数（1, 4, 9, 16, ...）
    
- 背包容量是整数 `n`
    
- 目标是**用最少的完全平方数装满背包**
    

**状态定义**：

- `dp[i]`表示组成整数 `i`所需的最少完全平方数的数量。
    

**状态转移方程**：

对于每个整数 `i`，我们遍历所有小于等于 `i`的完全平方数 `j*j`：

```
dp[i] = min(dp[i], dp[i - j*j] + 1)
```

其中 `dp[i - j*j] + 1`表示使用完全平方数 `j*j`后，组成剩余部分 `i - j*j`的最少数量加 1。

**初始化**：

- `dp[0] = 0`：组成 0 需要 0 个完全平方数
    
- 其他 `dp[i]`初始化为一个较大的数（如 `n+1`），因为最多可以用 `n`个 1 组成
    

**最终结果**：`dp[n]`

### 代码实现（带注释）

```
class Solution {
    public int numSquares(int n) {
        // dp[i] 表示组成整数 i 所需的最少完全平方数的数量
        int[] dp = new int[n + 1];
        
        // 初始化 dp 数组
        for (int i = 0; i <= n; i++) {
            // 初始化为一个较大的数，因为最多可以用 n 个 1 组成
            // 使用 n+1 是因为 dp[i] 最大值为 i（i 个 1），所以 n+1 是一个安全的较大值
            dp[i] = n + 1;
        }
        
        // 组成 0 需要 0 个完全平方数
        dp[0] = 0;
        
        // 遍历所有整数 i
        for (int i = 1; i <= n; i++) {
            // 遍历所有小于等于 i 的完全平方数
            for (int j = 1; j * j <= i; j++) {
                // 状态转移：使用完全平方数 j*j
                dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
            }
        }
        
        return dp[n];
    }
}
```

### 优化说明

代码中的初始化为 `n+1`是合适的，但也可以使用 `Integer.MAX_VALUE`或直接使用 `i`（因为 `dp[i]`最大为 `i`）。以下是更简洁的初始化方式：

```
class Solution {
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        
        // 初始化，dp[0] = 0，其他 dp[i] = i（最坏情况，由 i 个 1 组成）
        for (int i = 0; i <= n; i++) {
            dp[i] = i;
        }
        
        // 遍历所有整数
        for (int i = 1; i <= n; i++) {
            // 遍历所有完全平方数
            for (int j = 1; j * j <= i; j++) {
                dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
            }
        }
        
        return dp[n];
    }
}
```

### 复杂度分析

- **时间复杂度**：O(n√n)，其中 n 是给定的整数。外层循环 O(n)，内层循环 O(√n)。
    
- **空间复杂度**：O(n)，用于存储 dp 数组。
    

### 其他解法

本题还可以使用**数学方法**（四平方和定理）和**BFS（广度优先搜索）**​ 解决，但动态规划是最直观和通用的解法。
